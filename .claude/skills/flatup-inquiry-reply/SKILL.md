# flatup-inquiry-reply

## 目的
問い合わせに対する安全で温かい返信の下書きを作る。送信は人間。

## 手順
1. 問い合わせの意図を分類（料金 / 体験 / 日時 / 不安）。
2. 正本を確認：`canon/pricing_and_schedule.md`, `canon/membership_rules.md`, `canon/brand_voice.md`。
3. 日曜・祝日の体験は案内しない。平日枠を「候補」として提示。
4. 3案（丁寧版 / 標準版 / 短文版）を作る。雛形 `templates/inquiry_reply.md`。
5. 不明点は推測せず「確認事項」として残す。
6. 退会・休会・違約金・金銭は断定せず「担当より正式にご案内します」。

## 禁止
- 料金・日時の捏造。医療効果の断定。押し売り。自動送信。
