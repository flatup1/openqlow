# GYMSTORYs 特別編『そのちから、なにに使う？』— 二層演出設計(表: 子ども向け教則 / 裏: 親への手紙)

> この文書は、作品の「魂」を定義する正本です。映像・脚本・ナレーション・編集を担当するすべてのAI/人は、まずこれを読んでください。
> 上位ブランド正本(`brand-film-ep01/00_MASTER_CONTEXT.md`)と canon(`src/shared/canon.ts`)に従います。矛盾したらそちらが優先。

## 0. この作品がふつうの教則アニメと違うところ

この作品には、**2つの層**があります。

- **表の層(子どもが見るもの)**: かわいいマスコットが、怒ったときにどうするかを教える、やさしい教則アニメ。
- **裏の層(親の心に届くもの)**: 「大きくなっていく我が子へ、いま伝えておきたい大切なこと」を残す、**親から子への手紙**。

子どもは表の層で「怒っても、すぐ手を出さない」を学びます。
親は裏の層で、自分と我が子の“今しかない時間”を思い出します。

**作り手の心構え(スタッフだけが持つ、画面には決して出さない気持ち):**
> もし自分に残された時間がわずかだとしたら——
> 大きくなっていく娘に、何をひとつだけ残せるだろう。
> 強くなる方法ではなく、その強さを「どう使うか」。それを教えて逝きたい。

この気持ちを、**言葉にはせず、作品全体の誠実さとして滲ませます。** これが「琴線に響く」の正体です。

---

## 1. 絶対に守る倫理の線(最重要)

この二層構造は、まちがえると「不安商法」になります。FLAT UP GYMのブランドはそれを固く禁じています。次を厳守してください。

1. **「余命」「病気」「親がいなくなる」を、映像・音・字幕・広告文のどこにも出さない。** これは作り手の内なるコンパスであって、作品の題材ではない。
2. **親を脅さない。** 「今やらないと後悔する」「いなくなってからでは遅い」等は禁止。伝えるのは恐怖ではなく、**「今、そばにいられることの尊さ」**。
3. **子どもを泣かせる装置にしない。** 子どもは最後まで安心して見られる。悲しい顔で終わらせない。
4. **父親を主役にしすぎない。** 「父から娘へ」は作り手の気持ちの源。画面では**親を特定しすぎず**(手・背中・見守る気配だけ)、母親でも、祖父母でも、どんな保護者でも自分を重ねられる普遍性を保つ。
5. **説明しない。** 裏の層はナレーションの“余白”と映像の“間”で感じさせる。全部を言葉にした瞬間に安っぽくなる。

> ひとことで言うと——**「泣かせにいく」のではなく、「そっと隣に座る」作品にする。**

## 2. 二層の対応表(表で教え、裏で残す)

| 表の層(子どもへの教則) | 裏の層(親の心に残る手紙) |
|---|---|
| 強くなるのはこわくない | 君が挑戦する姿を、そばで見られるのが幸せだった |
| できなかったことができるようになる | 君の「できた！」を、ずっと覚えておきたい |
| 怒ったとき、すぐ手を出さない | いつか君がひとりで決める日、思い出してほしい |
| 深呼吸して、もう一回 | 迷ったら、一度立ち止まっていい |
| 出せる手を、自分で止める | 本当に強い人は、力を持っても振るわない人 |
| 強くなったから、優しくできる | その力は、だれかを守るためにある |
| はじめの一歩を、笑わない | 君の一歩を、パパはずっと笑わない |

表の左を子どもが受け取り、裏の右を親が受け取る。**同じ映像・同じ言葉で、両方に同時に届く**のが理想です。

## 3. ナレーション完成稿(裏の層の声=親の手紙)

声の方向性: 男性でも女性でもよい、低めで温かく、ささやきすぎない。**娘の未来に宛てて、静かに読む手紙**のように。力まない。泣かない。1行ごとに必ず“間”を置く。全部は使わず、映像に合わせて最小限に。

### 本編45秒版・ナレーション(道具たちの会話やインストラクターの台詞と重ねない箇所だけに置く)

> ねえ、聞いて。
> 強くなるのは、こわいことじゃないよ。
>
> できなかったことが、できるようになる。
> それを、そばで見ていられるのは——
> しあわせなことなんだ。
>
> いつか、ひとりで決める日がくる。
> 怒って、手が出そうになる日も。
>
> そのとき、思い出してほしい。
> 強いから、手を出さないんだって。
> 出せる手を、止められること。
> それが、いちばん伝えたかったこと。
>
> その力は、だれかを守るためにある。
>
> 大きくなった君へ。
> はじめの一歩を、パパは笑わない。
> ずっと、笑わないよ。

- 最終行「大きくなった君へ / ずっと、笑わないよ。」が、この作品の心臓。**“大きくなった君へ”**の一語だけで、時間の経過と「残しておきたい」気持ちが立ち上がる。病気も別れも一切言わずに、親の永遠の約束になる。
- 「パパ」は、ブランドの普遍性を優先するなら **「わたし」** に置き換え可(母親・祖父母も自分を重ねられる)。A/Bで検証してよい。ただし作り手は常に“父から娘へ”の温度で書く。

## 4. 「手」と「背中」の演出(裏の層を、言葉なしで運ぶ)

`brand-film-ep01` の EP01 から受け継ぐ2つの象徴を、本作の背骨にする。

- **小さな手 → 大きな手**: 冒頭、グローブをはめる小さな手。ラスト近くで、その手に一瞬だけ“少し大きくなった手”が重なり、すぐ戻る(1秒以内・説明的にしない)。=娘が成長していく時間。
- **親の背中・見守る気配**: 親は正面から映さない。**背中越し / 手だけ / 少し離れて見守る肩**で存在させる。うなずき一つ、タオルを渡す手一つ。=「そばにいる」こと自体がメッセージ。

この2つがあれば、ナレーションを削っても裏の層は伝わる。**映像だけで意味が通ることを最優先**(判断基準 §6)。

## 5. シーン別・二層の重ね方(45秒本編に沿って)

`GYMSTORYs 本編構成`(別紙の制作ブリーフ §11)を土台に、各シーンへ裏の層を仕込む。

| 秒 | 表(子どもが見る) | 裏の仕掛け(親に届く) |
|---|---|---|
| 0–5 心を整える | みんなで正座して深呼吸 | 親の手が、子の背にそっと添えられて離れる(見守りの気配) |
| 5–11 できた技 | ガード→右ストレート→戻す | 「そばで見ていられるのは、しあわせ」のナレーション |
| 11–17 怒り(赤い火) | 順番を抜かれて悔しい | 親は止めに入らない。少し離れて、信じて見ている背中 |
| 17–22 白い灯り | 「今、使う力かな？」 | ナレ「いつか、ひとりで決める日がくる」=親がいない未来への祈り |
| 22–28 手を止める | 拳を下ろし、深呼吸 | 「出せる手を、止められること」=残したい教えの核心 |
| 28–34 本当の強さ | 先生「自分の怒りに負けなかった」 | 小さな手に、大きな手が一瞬重なる(時間の経過) |

| 43–47 結論 | 笑顔でガード、パンチはしない | ナレ「大きくなった君へ。ずっと、笑わないよ」 |

## 6. 音の設計(裏の層を支える)

- 基本は `brand-film-ep01/06_SCRIPT_AUDIO.md` の方針(やわらかいピアノ・呼吸音・環境音、殴打音や威圧音は禁止)。
- **手紙のナレーションが入る箇所は、музыкを引いてほぼ無音に**。声と“間”だけで届ける。
- ラスト「ずっと、笑わないよ」の直後に、ピアノ最後の1音だけ残して消える(EP01エンドカードと同じ余韻設計)。
- **音楽で泣かせにいかない。** 無音の勇気を持つ。

---

## 7. エンドカード(表の導線・裏の余韻の両立)

- 表(行動): ロゴ+「はじめの一歩を、笑わない。」+「体験受付中」(または見学案内)。
- 裏(余韻): その上に置く1行 **「強くなった我が子が、優しくなる場所。」**(保護者向け中心コピー)。
- 料金/QR/LINEの扱いは canon と別紙ブリーフ §24 に従う。**感情の後に、静かな導線**。脅し文句は置かない。

## 8. この作品が成功した状態

- 子どもが見て「怒っても、すぐ手を出さないんだ」と自然に学ぶ。
- 親が見て、**言葉にできない何かで胸が温かくなり、今日、我が子をもう少しだけ抱きしめたくなる。**
- そのどちらにも、「FLAT UP GYMなら、この子の一歩を一緒に見守ってくれそう」という感覚が静かに残る。
- そして——誰も、不安で追い立てられていない。**共感 → 安心 → FLAT UP → やさしい導線**の順序が守られている。

> 表は、子どもへの教則。
> 裏は、大きくなっていく我が子へ、親が残しておきたい、たったひとつの願い。
> 「その力を、なにに使うか」。
> それだけを、静かに手渡す作品にする。
