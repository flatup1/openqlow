### 演技指示

- 全員、囁きすぎない小声。「閉店後の教室で先生同士が今日の生徒を語る」温度感
- 「……」は必ず間として演じる(詰めない)。台詞間には環境音だけの余白を残す
- バッグさんの「強くなったね」は、勝敗の強さではなく「怖さを少し越えたこと」への労い。褒めるというより、そっと認める
- 最後の「また明日、みんなに会えるかな」は寂しさではなく、楽しみにしている声で

## 4. 音楽・環境音・効果音の指示(成果物12)

### 音楽

- 楽器: ピアノ主体+後半に柔らかいストリングス薄く。打楽器・ドラムは使わない
- 構成:
- 0:00–0:12(夜): **音楽なし**。環境音のみで引き込む
- 0:12(昼転換): ピアノがぽつり、と入る(単音〜シンプルな分散和音、テンポ60前後)
- 0:19–0:22(うなずき): 音楽を絞りほぼ無音に。**この3秒が本作最大の「余白」**
- 0:22–0:31(一歩〜手): ピアノ再開、ストリングスが薄く重なる。盛り上げすぎない(サビを作らない)
- 0:34–0:45(夜〜ロゴ): 音量を落とし、最後はピアノ1音+残響で終わる
- 禁止: 感動系の壮大なビルドアップ、ボーカル曲、急な音量変化。「音楽だけで泣かせようとしない」が絶対原則

### 環境音

- 夜: 時計の針(S1で明瞭に、以降は微か)、空調の低いノイズ、建物の小さなきしみ、窓外の遠い夜の気配
- 昼: ジムの空気感(遠くの子どもの声・笑い声を小さく)、窓外の柔らかい環境音
- 帰り道: ドアの開閉音、外の夕方の環境音

### 効果音(すべて小さく・優しく)

| シーン | 音 |
|---|---|
| S2 | 革と布の擦れる小さな音(ミット・グローブが動く) |
| S2/S11 | チェーンの微かな軋み(バッグさんの揺れ) |
| S4 | 面ファスナーを留める音、グローブの革の音 |
| S5 | 子どもの小さな呼吸(吸って、止める) |
| S6 | ほぼ無音(空調のみ) |
| S7 | ミットに触れる「ポフッ」という軽い音(強い打撃音は禁止) |
| S9 | タオルの布の音、手が触れる気配 |
| S12 | 照明が落ちる小さな音(バチンではなく、ふっと消える音) |

### ミックス方針

- スマホスピーカー再生を基準にミックスする(小音量でも台詞が聞き取れること)
- 台詞 > 環境音 > 音楽 の優先順位。ラウドネスはSNS標準(-14 LUFS目安)で、CMのような音圧を稼がない
- 無音(に近い)区間を怖がらない。S6の余白がこの作品の心臓
