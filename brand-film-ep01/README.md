## 制作フロー(推奨手順)

1. `03_ART_BIBLE.md` のキャラクターシート用プロンプトで、道具キャラ・人物・ジム内装の**基準画像**を先に確定する
2. 基準画像を承認したら、`04_IMAGE_PROMPTS.md` で全シーンの静止画を生成する(共通ブロックを毎回貼る)
3. 静止画の一貫性チェック(`05_VIDEO_PROMPTS.md` の破綻防止チェックリスト)を通す
4. `05_VIDEO_PROMPTS.md` の Image-to-Video プロンプトで、1シーン=1動作の短いクリップを生成する
5. `07_EDIT_DELIVERY.md` のタイムラインどおりに編集し、`06_SCRIPT_AUDIO.md` の音・会話・字幕を載せる
6. `08_TESTING_METRICS.md` の A/B 設計で公開・計測する
