# 成果物1: 広告の最終コンセプト

## 作品タイトル

**FLATUP GYM BRAND FILM — EPISODE 01「今日一番頑張った子は誰？」**

## ログライン(1文)

夜、誰もいなくなったジムで道具たちが今日の子どもの話をする——何度もお母さんを振り返り、それでも最後にもう一歩踏み出した、あの子の話を。

## コンセプトステートメント

これは格闘技の広告ではない。**「今しかない親子の時間」を、ジムの道具たちの記憶として描く短編映画**である。

- 売るもの: キックボクシングの技術ではなく、「子どもの挑戦を、親が安心して見守れる時間」
- 語り手: 人間ではなく、その日を見ていた道具たち(ミット・グローブ・サンドバッグ)
- クライマックス: 大技でも勝利でもなく、「不安で止まった子が、親のうなずきひとつで、もう一度ミットに触れる」瞬間
- 余韻: 「また明日、みんなに会えるかな」——ジムが人を待っている、という優しさ

## なぜ「道具が語る」のか(設計意図)

1. **売り込みの声を消せる**: ジム側の人間が語ると宣伝になる。道具の会話なら、第三者が見ていた記憶として自然に伝わる
2. **恐怖の記号を反転できる**: ミットやサンドバッグは「怖い格闘技」の象徴。それが子どもを優しく見守る存在として描かれることで、「怖い格闘技ジム」という先入観そのものが崩れる
3. **シリーズの器になる**: 道具たちは毎晩違う誰かの話ができる(女性編・大人編・シニア編へそのまま展開可能)
4. **冒頭1〜3秒の静かな違和感**: 「夜のジムで何かが動いた」は、大声やテロップに頼らずスクロールを止められる

## 感情設計(視聴者の心の動き)

| 段階 | 視聴者に起きること |
|---|---|
| 静けさ | 「広告っぽくない」と手が止まる |
| 小さな不思議 | 道具が動く。怖くない、むしろ愛おしい |
| 子どもの記憶 | 「あの子」に自分の子ども(または昔の自分)を重ね始める |
| 不安と、親のうなずき | 「うちの子もこうだった」——視聴者自身の記憶が再生される |
| もう一歩 | 勝敗ではない達成に、静かに胸が動く |
| 手・帰り道 | 「この時間は今しかない」と気づく |
| 夜へ戻る・余韻 | ジムが子どもを覚えていてくれた、という安心 |
| ロゴ・導線 | 「FLATUP GYMなら、見守ってくれるかもしれない」が静かに残る |

## 成功の定義

- 見終わった親が、今日、子どもを少し抱きしめたくなること
- 「怖くなかった」「うちの子にもできそう」「泣いた」というコメントが自然に生まれること
- その感情の延長線上で、プロフィール遷移 → LINE登録 → 体験予約が発生すること

## この広告でやらないこと(正本§27の運用宣言)

- 技術・強さ・試合・勝敗を見せない(パンチは「軽くミットに触れる」1回のみ)
- 親の不安・罪悪感を刺激するコピーを使わない
- 料金・キャンペーン訴求をしない(CTAは「体験受付中」等の静かな案内のみ)
- 画面を文字で埋めない(同時表示テキストは最大2行)
- 既存有名アニメ作品の模倣をしない(オリジナルの家族向け3Dアニメーションとして設計)

## 二層構造の確認

- **第一話の感情の中心**: 親子(6歳前後の子どもと、見守る母親)
- **ブランド全体の中心**: すべての初心者の最初の一歩(第一話のラストコピーは親子限定の言葉ではなく「人生の最初の一歩に寄り添う場所。」を主軸にし、女性・大人・シニアを排除しない)
