# OPENQLOW 言葉じてん

出てくる専門用語を、中学生むけにやさしく説明します。

| 言葉 | やさしい意味 |
| --- | --- |
| **ドラフト / 下書き** | まだ送っていない、下書きの文章。OPENQLOWはこれを作るだけ。 |
| **SNS** | X（旧ツイッター）、Instagram、Threads などのネットの発信の場。 |
| **CRM** | お客さんの「名簿・カルテ」。だれにいつ連絡したかを記録するしくみ。 |
| **見込み客** | これから入会してくれそうなお客さんのこと。 |
| **追客（ついきゃく）** | 「その後どうですか？」と、もう一度連絡して追いかけること。 |
| **AIKA** | OPENQLOWが返事を書くときの“キャラ・口調”の名前。やさしく丁寧。 |
| **正本（しょうほん）／canon** | 「これが正しい」と決められた事実（料金など）。AIが勝手に変えない大もと。 |
| **承認（しょうにん）** | 人間が「これでOK」と許可すること。 |
| **LINE Bot / Webhook** | LINEのメッセージを受け取る“自動の窓口”のしくみ。 |
| **スケジューラ / systemd / timer** | 「毎朝7時に動かす」のような“目覚まし時計”のしくみ。 |
| **VPS** | インターネット上で24時間動く、借りているコンピュータ（サーバー）。 |
| **nginx** | 来た人を正しい入口へ案内する“受付の交通整理”役。 |
| **Obsidian / Vault** | メモを[[obsidian-vault\|リンクでつなげて]]管理するアプリと、その保存箱。 |
| **Phase 1（フェーズ1）** | 「第1段階」。今はまだ“投稿しない・準備だけ”の段階という意味。 |
