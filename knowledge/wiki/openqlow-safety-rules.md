# OPENQLOW のまもるルール（安全のやくそく）

このAIには「ぜったいに守るやくそく」があります。
かんたんに言うと **「AIは下書きまで。最後は人間が決める」** です。

## 6つのやくそく
1. **勝手にSNSへ投稿しない。** 作るのは“下書き”だけ。
2. **人間のOKがいる。** LINEで `OK 番号` ときちんと言われたものだけ進む。
3. **予約・公開はできないようにしてある。** Phase 1 ではプログラム自体が断る（物理的に拒否）。
4. **料金やスケジュールを勝手に変えない。** 決まった正しい値（正本）だけを使う。
5. **おおげさ・きつい表現はしない。**「必ず痩せる」みたいな言い方や、体型を否定する言葉は使わない。
6. **ひみつ（APIキー）や個人情報を書き残さない。** Git や README やメモに残さない。

## どうしてこんなに慎重なの？
SNSの投稿やお客さんへの連絡は、**まちがえると取り消せない**からです。
だから「考える・下書きする」のはAI、**「送る・決める」のは人間**、と役割を分けています。
これは [[second-brain|第二の脳]] の知識ベースでも同じ考え方です（ひみつは書かない）。
