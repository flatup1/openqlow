# FLATUP 正本FAQ

> **両AI（AIKA / OPENQLOW）は、料金・スケジュール・クラス・住所・体験を聞かれたら必ずここだけを引く。**
> ログでは同じ質問に違う答えを返す不整合・推測があった。それを止めるための単一正本。
> ✅=正本確定（コード `FLATUP_INFO` 由来）／⚠️=要オーナー確認（実運用で値がブレていた。確定前に推測で答えない）。

## 料金 ✅
- 初回体験: **500円**
- 2回目以降ビジター: **3,000円**
- 月額: キッズ **7,700円** / 女性 **8,800円** / 男性 **9,900円**
- 入会金: **10,000円**
- 親子割: **親子合計で月-¥500** ✅（オーナー確定 2026-06-25）
- 紹介特典: **紹介でお互いにFLATUPバンテージ進呈** ✅（オーナー確定 2026-06-25）

## 体験予約の枠 ✅
- キッズ: 火曜・木曜 18:00、土曜 13:00
- レディース: 土曜 14:00
- 男性の体験: 火曜・木曜・土曜（男性インストラクター在籍）、または平日 19:00 以降
- 女性の体験: 月曜・水曜・土曜
- 体験不可: **日曜・祝日は原則不可**

## 営業時間 ✅（オーナー確定 2026-06-25）
- 平日 **10:00〜12:00 / 18:00〜20:00**、土曜はクラス制、**日曜・祝日は休み**。

## クラス（種目）✅（オーナー確定 2026-06-25）
- **ボクシング／キックボクシング／ムエタイ／寝技／レスリング／総合格闘技(MMA)／ブラジリアン柔術／キッズ／レディース**（9種）。
- ムエタイは**正式クラスに含む**。

## 住所・アクセス ✅（オーナー確定 2026-06-25）
- 住所: **成田市土屋516-4 2F（百香亭の上）**
- 駐車場: 専用駐車場あり
- 最寄り駅: **成田駅**
- 行き方: **成田駅からイオンモール行きのバス → イオンのバス停で下車、徒歩約5分**

## 持ち物 ✅
- 動きやすい服・タオル・水（飲み物）。体験時はグローブ/ミット貸出あり＝**初回は購入不要**。
- グローブ＋レガースのセット: **11,000円**（購入は任意）✅（オーナー確定 2026-06-25）

## 体験前チェック（AIKAの良い資産・維持）
- Q: 対象者 / 年代 / きっかけ / なりたい姿 / 不安 / スパーリング希望 / 流入元 / 怪我・体調 → 安全注意の同意。
- これは継続。属性は[[../sources/20260625-aika-line-log-summary|CRM]]へ。

## 使い方ルール
1. ✅ は即答してよい。⚠️ は**断定せず**「確認してご案内します」と返し、オーナーに確認 → 確定したらこのページを ✅ に更新。
2. 料金改定・スケジュール変更時は、コード `src/generators/shared.ts` の `FLATUP_INFO` と**このページの両方**を更新。
3. 推測で答えた場合は必ず「推測」と明記（AGENTS.md 規則6）。
