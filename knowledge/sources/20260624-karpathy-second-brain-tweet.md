# Karpathy式 第二の脳をClaude Codeで作る

> 出典: https://x.com/textocriativo/status/2069777875184599409 ／ 取り込み日: 2026-06-24
> （X本体はこの環境のネットワークポリシーで直接取得不可。本文は依頼者が貼り付けたもの。）

OpenAIの共同創設者であるAndrej Karpathyが共有したシンプルなアイデアが、すでに1600万回以上の視聴を記録しています：AIをコードを書くためだけに使うのをやめて、第二の脳を構築するために使い始めましょう。

Claude Codeをフォルダに接続し、どんな情報源でも追加してください — 記事、書き起こし、PDF、メモ — すると、それらすべてを読み込み、整理し、つながりを作成し、あなたの知識を生きているウィキに変えてくれます。

システムに情報を追加するほど、それはより賢く、より役立つものになります。

仕組みはこうです：

→ Obsidianをインストールし、Vaultを作成してClaude Codeで開く

→ Karpathyが作成したウィキファイルの貼り付けを求め、Claudeに設定を依頼する

→ Claudeは3つの構造を作成します：生のソース用のフォルダ、ウィキページ用のフォルダ、そしてすべてを管理するためのCLAUDE.mdファイル

→ ソースフォルダにどんなコンテンツでも追加して、「これを分析して整理して」と言う

→ 次に、保存されたどんな情報についても質問する — すべてがつながって検索可能になります

設定には約5分しかかかりません。

そして最高なのは：もうゼロから会話を始める必要がなくなります。

後で試すために、この投稿を保存してください。
