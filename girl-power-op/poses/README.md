# poses フォルダ — 本物のアニメにする場所

ここに「ポーズちがい」のキャラ画像を置くと、シーンに合わせて絵が切りかわる
**本物のアニメ(リミテッドアニメ方式)** になります。何も置かなければ、
今まで通り `chara.png` 1枚で動きます。

## ファイル名の決まり(名前でシーンが決まる)

| ファイル名 | いつ表示される? |
|---|---|
| `idle.png` | ふだんの構え(カット1・3など) |
| `punch1.png` | パンチの瞬間その1(カット2で交互に出る) |
| `punch2.png` | パンチの瞬間その2 |
| `jump.png` | ジャンプ中(カット4)と空を飛ぶシーン(カット5) |
| `win.png` | 最後の決めポーズ(カット6) |

- ぜんぶそろえなくてOK。無いポーズは `chara.png` で代用されます
- 画像は **同じキャラ・同じくらいの大きさ・縦長** でそろえると自然に見えます
- 画像生成AI(Midjourney / NijiJourney など)で「同じキャラの別ポーズ」を
  作るときは、`chara.png` を参照画像にして
  「same character, throwing a left punch」のように頼むと作りやすいです

## 使い方

画像を置いたら、いつも通り実行するだけ:

```bash
npm run video            # 横型
npm run video:vertical   # 縦型
```

`render.js` がこのフォルダを自動でチェックして、見つけた画像を使います。
