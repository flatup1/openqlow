# GIRL POWER! OP メーカー

キャラクターの画像 **1枚** から、30秒のアニメOP風動画(BGM付き・MP4)を自動で作るツールです。
プログラムがブラウザの中に絵を描いて、それを1コマずつ写真に撮って、動画にまとめます。

![完成イメージ: キャラが登場→パンチ→クローズアップ→ジャンプ→空→ロゴでフィニッシュ]

---

## しくみ(どうやって動画ができるの?)

動画の正体は **パラパラまんが** です。

1. `index.html` の中に「**時刻 t 秒のときの絵を描く**」プログラムが入っています
   (例: 「6.2秒の絵を描いて」と頼むと、パンチしてハートが飛び散る瞬間の絵を描く)
2. `render.js` が撮影係のロボットです。ブラウザを自動操縦して、
   時刻を 1/30 秒ずつ進めながらスクリーンショットを **900枚** 撮ります
   (30秒 × 1秒あたり30コマ = 900コマ)
3. BGM もブラウザの中で作ります。「何秒にドラム」「何秒にメロディの音」と
   楽譜をプログラムで書いて、音の波形を計算して WAV ファイルにします
4. 最後に **ffmpeg**(動画をつなぐ道具)が、画像900枚 + BGM を 1本の MP4 にまとめます

つまり「AIが動画を生成する」のではなく、**ぜんぶ計算で描いている**ので、
何回作っても同じ結果になり、好きなところを自由に改造できます。

## 用意するもの

| 道具 | 何のため? | 入れ方 |
|---|---|---|
| **Node.js** (LTS版) | プログラムを動かすため | https://nodejs.org からダウンロード |
| **ffmpeg** | 画像900枚を動画にまとめるため | Windows: `winget install ffmpeg` / Mac: `brew install ffmpeg` |

入っているか確かめるには、ターミナル(コマンドプロンプト)で:

```bash
node -v     # v20.xx などと出ればOK
ffmpeg -version   # バージョンが出ればOK
```

## 作り方(コマンド3つ)

このフォルダ(`girl-power-op`)をターミナルで開いて、順番に打ちます。

```bash
# 1. 部品(Playwright = ブラウザ自動操縦ライブラリ)を入れる
npm install

# 2. 自動操縦用のブラウザ(Chromium)を入れる ※初回だけ
npx playwright install chromium

# 3. 動画を作る!(3〜5分かかります)
npm run video
```

終わると同じフォルダに **`girl_power_op.mp4`** ができています。ダブルクリックで再生してみよう。

## 自分のキャラに変える

このフォルダの **`chara.png` を好きな画像に差しかえるだけ**です(ファイル名は `chara.png` のまま)。

- **縦長の画像**(スマホの写真のような比率)がいちばんキレイに収まります
- キャラが真ん中に大きく写っていて、顔が上半分にあるとクローズアップのカットが決まります

## 改造してみよう

`index.html` をメモ帳やVS Codeで開いて、キーワードで検索して数字や文字を変える → もう一度 `npm run video` で反映されます。

| 変えたいこと | 検索するキーワード | 説明 |
|---|---|---|
| 画面に出る文字 | `popText(` | `'GIRL POWER!'` などを好きな言葉に(日本語もOK) |
| 動画の長さ | `DUR = 30` | 秒数。`render.js` 側の `DUR = 30` も同じ数字にする |
| カットの切りかわる時間 | `C1 = 0, C2 = 4.5` | 各カットの開始秒。C1→C6 の順 |
| 色(ピンクや黄色) | `palette` | `#ff5fa2` のようなカラーコードを変える |
| 曲の速さ | `60 / 160` | 160 が BPM(1分あたりの拍数)。大きいほど速い曲に |
| メロディ | `riffA` `riffB` | 数字の並びが音の高さ。変えると別のメロディに |

失敗しても大丈夫。git で管理されているので `git checkout -- index.html` でいつでも元に戻せます。

## 6つのカットの構成(絵コンテ)

| カット | 時間 | 内容 |
|---|---|---|
| 1 | 0.0〜4.5秒 | 星形の穴が開いてキャラ登場(ピンクの放射線背景) |
| 2 | 4.5〜9.0秒 | ビートに合わせてパンチ連打。ハートが飛び散り「POW!」 |
| 3 | 9.0〜13.5秒 | 顔のクローズアップ + 集中線 + 「FIGHT!」 |
| 4 | 13.5〜18秒 | ぴょんぴょんジャンプ(着地でぐにゃっとつぶれる演出付き) |
| 5 | 18〜22.5秒 | 青空をふわふわ浮遊。ハートと星が舞う |
| 6 | 22.5〜30秒 | 紙吹雪 + 「GIRL POWER!」ロゴでフィニッシュ |

## こまったとき

- **`ffmpeg` が見つからないと言われる**
  → ffmpeg をインストールしてターミナルを開き直す。それでもダメなら
  `FFMPEG=/ffmpegの場所 npm run video` のように場所を直接教える
- **`npx playwright install chromium` が失敗する**
  → ネット接続を確認してもう一度。会社や学校のネットだとブロックされることがあります
- **動画が真っ黒 / キャラが出ない**
  → `chara.png` がこのフォルダにあるか、ファイル名が正しいか確認
- **音がずれている気がする**
  → `index.html` と `render.js` の `DUR` の数字がそろっているか確認
