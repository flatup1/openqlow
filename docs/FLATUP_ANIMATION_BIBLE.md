# FLATUP ANIMATION BIBLE 2027 — v1.0（正式版）

世界一初心者に優しい格闘技ジムの、物語・映像・キャラクター制作基準

作成: 2026-07-11（JIN正本）
上位文書: `FLATUP_ANIMATION_PROJECT_2027.md`（ビジョン）／`FLATUP_ANIMATION_ENGINE_2027.md`（技術ロードマップ）

## 1. この資料の目的

FLATUP Animation Bibleは、FLATUP GYMのアニメーションを制作する際に、すべての判断基準となる設計書です。

AI、エンジニア、アニメーター、イラストレーター、動画編集者など、誰が制作しても、キャラクターの性格・映像の雰囲気・色・光・動き・セリフ・物語・ブランドメッセージが崩れない状態を作ります。

この資料の最優先目的は、映像を豪華にすることではありません。

**FLATUP GYMの優しさを、すべての場面で守ること。**

これが最も重要な役割です。

## 2. ブランドの中心思想

**世界一初心者に優しい格闘技ジム**

FLATUP GYMは、最初から強い人のためだけの場所ではありません。

怖い人。自信がない人。運動が苦手な人。何かを始めることに不安がある人。過去に失敗した人。周囲と比べて落ち込んでいる人。

そのような人が、安心して最初の一歩を踏み出せる場所です。

## 3. ブランドの約束

FLATUP GYMの物語では、次の約束を守ります。

- **怒鳴らない** — 失敗したキャラクターを、怒鳴って動かす演出は使用しません。
- **笑わない** — 不器用さや失敗を、見下す笑いにしません。
- **比べない** — 他のキャラクターより弱いことを、恥として描きません。
- **急かさない** — すぐに成長することを求めません。
- **置いていかない** — 一人だけ取り残される物語で終わらせません。
- **挑戦を称える** — 結果よりも、挑戦した事実を大切にします。

## 4. 作品の中心メッセージ

> **はじめの一歩を、笑わない。**

この言葉を、すべての作品の判断基準にします。迷ったときは、

**「この場面は、初めて挑戦する人を安心させるか。」**

で判断します。

- 格好よく見えても、初心者を怖がらせる場合は採用しません。
- 笑える場面でも、失敗した人を傷つける場合は採用しません。
- 感動的でも、無理に努力を押しつける場合は採用しません。

## 5. 作品が届ける感情

FLATUPのアニメーションでは、次の感情を順番に設計します。

1. **興味** — 「このキャラクターは何者だろう」「夜のジムで何が始まるのだろう」と感じてもらいます。
2. **共感** — 主人公の不安や緊張を描きます。視聴者が「自分と少し似ている」と思えることが重要です。
3. **安心** — 主人公が失敗しても、誰も責めません。仲間が手を差し伸べます。ここでFLATUP GYMの価値を伝えます。
4. **小さな挑戦** — 主人公は、完璧を目指しません。一歩だけ進みます。一発だけパンチを打ちます。
5. **喜び** — 大勝利ではなく、「自分にもできた」という小さな喜びを描きます。
6. **優しさの継承** — 主人公は、次に挑戦する人へ手を差し伸べます。自分が受け取った優しさを、次の人へ渡します。
7. **行動** — 作品を見た人が「私も体験してみようかな」と思える形で終わらせます。

## 6. 世界設定

### 舞台

千葉県成田市にあるFLATUP GYM。白い床。緑のマット。明るいサンドバッグ。観葉植物。窓から入る自然光。現実のジムの特徴を、アニメの世界にも反映します。

### 昼の世界

昼間は、人間が練習しています。キャラクターたちは道具やマスコットとして静かに見守っています。

### 夜の世界

ジムの扉が閉まり、照明が消えると、キャラクターたちは動き始めます。彼らは夜の間に、

- 練習する / 失敗する / 助け合う
- 昼間に来た人について話す
- 新しい挑戦を考える
- 翌日の初心者を迎える準備をする

という生活をしています。

### 朝の世界

朝日が差し込むと、キャラクターたちは元の場所へ戻ります。人間は彼らが動いていることを知りません。ただし、キャラクターたちの優しさは、ジムの空気として残っています。

## 7. 世界観の絶対ルール

### 悪役を中心にしない

FLATUPの物語に、分かりやすい悪人は必要ありません。主人公が向き合うものは、**不安・自信のなさ・失敗への恐怖・周囲の目・過去の苦手意識**です。

### 暴力を快楽として描かない

パンチやキックは、誰かを傷つけるためではありません。練習、成長、挑戦、自己表現として描きます。

### 勝者だけを称えない

勝ったキャラクターだけを中心にしません。挑戦したキャラクター。仲間を支えたキャラクター。最後まで続けたキャラクター。これらも同じように称えます。

### 強さを見た目だけで決めない

身体が大きいことや、技が上手いことだけを強さとして描きません。**謝れる・助けられる・待てる・何度でも挑戦できる・他人に優しくできる**ことも強さとして扱います。

## 8. 主人公設計

### 仮称

**フラッピー**（名称は今後変更可能ですが、役割と人格は固定します）

### 主人公の役割

視聴者が自分を重ねる存在です。最初から完成されたヒーローではありません。

### 性格

少し怖がり / 素直 / 優しい / 周囲を気にしやすい / 失敗すると落ち込む / それでも小さく再挑戦する / 仲間から受けた優しさを忘れない

### 長所（技術や力ではありません）

- 人の気持ちに気づける
- 失敗した人に寄り添える
- 小さな変化を喜べる
- 自分が怖かった経験を覚えている

### 弱点

- 他人と比べてしまう
- 失敗を恥ずかしいと思う
- 最初の一歩に時間がかかる
- 大きな音や強い相手に緊張する

### 成長の方向

最強の選手になることが最終目的ではありません。**挑戦する人の隣に立てる存在になること。** これが主人公の本当の成長です。

## 9. 主人公の外見設計

- **シルエット**: 遠くから見ても識別できる、単純で強い形。ただし既存作品の特徴的な頭身や目の形を再現しない
- **頭身**: かわいさだけを優先して手足を極端に短くしない。パンチ・キック・構え・歩行が読み取れる体型
- **顔**: 大きく感情が読める目 / 独自の瞳形状 / 小さな表情変化が伝わる眉 / 子どもっぽすぎない口元 / 横顔でも成立する造形
- **グローブ**: 少し大きめ。最初は「自分には大きすぎる挑戦」を象徴する。物語が進んでもグローブ自体は小さくならず、主人公が少しずつ扱えるようになることで成長を表現する
- **配色**: 白・緑・蛍光ピンク・蛍光イエロー・温かいオレンジ（FLATUP GYMの実際の空間とつながる色）。1体にすべての色を使用せず、**主色・補助色・アクセント色の3段階**に分ける

## 10. キャラクターの表情ルール

表情は大げさな顔だけで作りません。目線・まばたき・眉・口・頭の角度・肩・手の位置・足の向き・呼吸・身体の重心を組み合わせます。

- **緊張**: 肩が少し上がる / グローブを胸に近づける / 目線が定まらない / 足幅が狭い / 呼吸が浅い
- **不安**: 出口を見る / 周囲を確認する / 一歩下がる / 口を閉じる / 身体を小さく見せる
- **決意**: 急に険しい顔にしない / 深呼吸する / 足を半歩前に出す / 相手ではなく目標を見る / 眉が少し上がる
- **安心**: 肩が下がる / 呼吸がゆっくりになる / 目がやわらかくなる / 小さく笑う / 仲間との距離が近くなる
- **喜び**: 毎回大きく飛び上がらない。小さな笑顔・驚いた目・グローブを見つめる・仲間と目を合わせる、という静かな喜びも大切にする

## 11. 仲間キャラクターの役割

キャラクターは、見た目の違いだけで増やしません。物語上の役割を持たせます。

- **案内役**: 主人公に答えを教えるのではなく、隣で待てるキャラクター。命令口調は使わない
- **元気な仲間**: 失敗を恐れず、場を明るくする。ただし主人公の不安を軽く扱わない
- **静かな仲間**: 言葉より行動で優しさを示す。バンテージを渡す。倒れた道具を直す。主人公の隣に座る
- **努力家の仲間**: 最初から上手い天才ではない。何度も繰り返す姿を見せる
- **新しい初心者**: 物語後半に登場し、主人公の成長を示す。主人公が以前の自分を見るような存在

## 12. 色彩設計

色は、かわいさだけでなく感情を伝えるために使います。

- **安心**: 柔らかい白 / 明るい緑 / 温かい黄色 / 低いコントラスト
- **緊張**: 青みのある夜 / 広い影 / 主人公の周囲だけ少し暗い / 背景との距離を感じる配色
- **孤独**: 完全な黒にはしない。FLATUPの世界には、暗い場面でも必ず小さな温かい光を残す
- **挑戦**: 少しずつ暖色を増やす。決意した瞬間に急激な赤や炎を使うのではなく、朝日に近い自然な温度変化で表現する
- **成長**: 金色の光だけに頼らない。仲間との距離、表情、姿勢、画面の広がりも同時に変える

## 13. ライティング設計

- **夜**: 月明かりと、非常灯や街灯のわずかな光。怖いホラー空間にはしない
- **練習場面**: キャラクターの目と表情が常に読み取れるように。顔が完全な影に入らないようにする
- **感情場面**: 光を派手に動かしすぎない。主人公の表情や呼吸を見せるため、静かな光を使う
- **朝**: 朝日は希望の象徴だが、過度に神々しくしない。新しい一日が自然に始まる温かさを目指す

## 14. カメラワーク

- **主人公が不安な場面**: 少し広めの画角 / 主人公を画面の端に置く / 周囲を大きく見せる / 強いキャラクターを必要以上に巨大化しない
- **仲間が寄り添う場面**: 同じ目線の高さ / ゆっくり距離を縮める / 上から見下ろさない / 主人公と仲間を同じ画面に入れる
- **挑戦する場面**: 極端な高速ズームや激しい揺れを避ける。深呼吸 → 足 → グローブ → 目 → サンドバッグ、という順番で気持ちと動作をつなげる
- **成功する場面**: パンチが当たった瞬間だけでなく、**その直後の主人公の表情**を最も大切にする

## 15. 動作設計

### 基本原則

動作は、技ではなく感情から始まります。パンチを作る場合でも、

1. 何を感じているか
2. 何を見ているか
3. なぜ動くのか
4. 動いた後に何が変わるか

を決めてから制作します。

### パンチの基本構成

```text
guard → look_target → breathe_in → anticipation → contact → followthrough → recovery → reaction
```

### 転倒の基本構成

```text
miss → lose_balance → fall → sit → check_others → embarrassed → hand_reach → standup
```

### 優しさの基本構成

```text
notice → pause → approach → sit_beside → look_same_direction → hand_reach → small_smile
```

## 16. 「間」のルール

FLATUPアニメーションでは、動き続けることを品質とは考えません。感情が伝わるための静止や沈黙を使います。

**良い間**: パンチを打つ前の深呼吸 / 転んだ後に周囲を見る時間 / 仲間がすぐ励まさず、隣に座る時間 / 初めて成功したことを理解する時間 / 新しい初心者へ近づく前の迷い

**悪い間**: 意味なく画面が止まる / テンポが遅いだけ / セリフを待っているだけ / 感情の変化がない / 同じ表情が長く続く

## 17. セリフ設計

短くします。説明しすぎません。正しいことを教えるより、安心できる言葉を選びます。

**使用したい言葉**:
一緒にやってみよう / 今日は一回だけでいいよ / 怖いままでも大丈夫 / 最初はみんな同じだよ / 上手じゃなくていいよ / できたね / 急がなくていいよ / ここにいていいよ

**避ける言葉**:
根性を出せ / 弱い / できるまでやれ / 泣くな / 男らしくしろ / 本気ならできる / 努力が足りない / 他の人はできている / 逃げるな

**セリフの優先順位**: ① 行動で見せる → ② 表情で見せる → ③ 音で見せる → ④ 必要な場合だけ言葉を使う

## 18. ユーモアのルール

FLATUPの作品には、温かい笑いを入れます。

**良いユーモア**: 大きすぎるグローブに振り回される / バンテージが長すぎて絡まる / サンドバッグが少しだけ動いて本人が驚く / 元気な仲間も実は転んでいる / 道具たちが夜中にこっそり準備する

**避けるユーモア**: 失敗者を集団で笑う / 容姿を笑う / 運動能力を笑う / 泣いている人を笑う / 初心者を迷惑な存在として扱う / 怪我を笑いにする

## 19. 音響設計

音を増やして豪華にするのではなく、世界への没入を深めます。

**基本環境音**: 時計の秒針 / 遠くを走る車 / 風 / 雨 / 鳥 / マットを踏む音 / 革が擦れる音 / バンテージを巻く音 / サンドバッグが揺れる音 / 呼吸 / 小さな足音

**パンチ音**: 爆発音のようにしない。柔らかいサンドバッグにグローブが当たる、少し低くて温かい音にする

**成功音**: 鐘やファンファーレを毎回使用しない。静かなピアノの一音・仲間の小さな拍手・サンドバッグが戻る音で成功を表現する

## 20. 音楽設計

- **導入**: 木製のおもちゃを思わせる音。小さなピアノ。優しい弦
- **不安**: 楽器数を減らす。旋律を止め、環境音と呼吸を聞かせる
- **挑戦**: 少しずつ音が加わる。急に壮大な音楽に変えない
- **成功**: 音量を上げるより、**旋律が完成することで成長を表す**
- **ラスト**: 温かさを残しながら、次の一歩を感じる余白を作る

## 21. 脚本構造

基本的な短編は次の構造で制作します。

1. **日常** — 主人公がいる世界を見せる
2. **不安** — 主人公が挑戦したいが、一歩を踏み出せない
3. **失敗** — 最初の挑戦がうまくいかない
4. **優しさ** — 仲間が責めずに寄り添う
5. **再挑戦** — 主人公自身が、小さくもう一度動く
6. **小さな成功** — 完璧ではない成功を描く
7. **優しさの継承** — 主人公が次の誰かを支える
8. **現実への接続** — 視聴者自身の体験予約へつなげる

## 22. 15秒映像の原則

サイトのファーストビューでは、すべてを説明しません。

**必須要素**: 主人公の不安 / 一度の失敗 / 仲間の優しさ / 小さな成功 / CTA

**推奨構成**:

| 秒 | 内容 |
|---|---|
| 0〜3 | 夜のジムで主人公が目を覚ます |
| 3〜6 | パンチを失敗する |
| 6〜9 | 仲間が手を差し出す |
| 9〜12 | 小さなパンチが成功する |
| 12〜15 | メッセージと体験予約を表示する |

**表示メッセージ**:
> 怖いままでも、大丈夫。
> はじめの一歩を、笑わない。

**CTA**: 500円体験を予約する

## 23. 30秒広告の原則

広告では最初の3秒で、視聴者自身の不安に触れます。

- **冒頭例**: 「格闘技ジムは、怖いと思っていませんか？」
- **中盤**: 主人公の失敗と仲間の反応を見せる
- **終盤**: FLATUP GYMでは、失敗しても笑われないことを伝える
- **CTA**: 世界一初心者に優しい格闘技ジム 成田市・初回体験500円

## 24. サイト実装ルール

- **自動再生**: 音声は自動再生しない。映像のみを静かに再生する
- **再生制御**: 画面外では停止 / 低スペック端末では軽量版へ切り替え / 通信が遅い場合は静止画を表示 / ユーザーが停止できるボタンを用意 / 動きを減らすOS設定（prefers-reduced-motion）に対応
- **モバイル**: 主人公の表情とCTAが小さくならない構図。横長映像の単純な切り取りではなく、縦画面用の構図を別に設計する
- **読み込み**: アニメーションによってサイト表示が遅くならないことを最優先。映像が読み込まれなくても、ブランドメッセージと体験予約ボタンは表示する

## 25. 生成AIの使用ルール

- **既存LoRA**: 既存作品を学習したLoRAは、研究・試作・技術検証のみに使用する。本番の公式素材は、FLATUP独自デザインへ移行する
- **オリジナル化**: 顔・目・髪型・体型・衣装・配色・ロゴ・グローブ・世界設定・性格・名前を FLATUP 独自のものとして固定する
- **一貫性チェック**（画像生成のたびに確認）: 同じ顔か / 同じ体型か / 同じ衣装か / 同じ色か / グローブの形が同じか / ロゴが変形していないか / 左右が入れ替わっていないか / 表情が場面に合っているか
- **禁止事項**: 既存キャラクター本人の使用 / 既存ロゴの使用 / 特徴的な衣装の再現 / 既存作品の構図をほぼそのまま複製 / 公式コラボと誤認される表現 / 特定作品名を正式タイトルや広告コピーに使用

## 26. FLATUP専用LoRAの学習基準

- **学習画像**: 推奨60〜100枚。ただし枚数より**統一性**を重視
- **必須画像**: 正面 / 左右斜め / 左右側面 / 背面 / 上から / 下から / 表情集 / 構え / パンチ / キック / 歩行 / 転倒 / 座る / 手を差し出す / 仲間を見る
- **学習からの除外**: 指が破綻 / 顔が別人 / 衣装が異なる / ロゴが変形 / 体型が大きく違う / 既存作品に近すぎる / 背景に不要な文字がある
- **トリガーワード**: FLATUP独自の文字列。例: `flatup_hero` / `flappy_flatup`

## 29. 品質確認チェックリスト

**ブランド**: FLATUPの優しさが伝わるか / 初心者を怖がらせないか / 挑戦を押しつけていないか / 失敗者を笑っていないか / 勝敗だけを価値にしていないか

**キャラクター**: 顔が統一されているか / 体型が統一されているか / 配色が統一されているか / グローブが破綻していないか / 表情と感情が一致しているか

**アニメーション**: 動く理由があるか / 動作前の予備動作があるか / 動作後の反応があるか / 速すぎないか / 静止する場面に意味があるか

**映像**: スマートフォンでも表情が読めるか / 文字が見やすいか / CTAが分かるか / 背景が主張しすぎていないか / 照明が暗すぎないか

**マーケティング**: 誰向けの映像か明確か / 見た後の行動が明確か / 体験予約につながるか / ブランドの記憶が残るか / FLATUP GYMである必要性があるか

## 30. 成功指標

作品の評価は、映像の美しさだけで判断しません。

- **視聴指標**: 再生開始率 / 3秒視聴率 / 15秒到達率 / 完了率 / 再視聴率
- **行動指標**: 体験予約ボタンのクリック率 / LINE追加率 / 初心者向けページへの移動率 / 体験申込数 / 入会数
- **感情指標**（体験者やサイト利用者へ確認）: 怖くなさそうと感じたか / 初心者でも大丈夫だと思ったか / 自分にもできそうだと思ったか / 子どもを通わせたいと思ったか / 他の格闘技ジムとの違いを感じたか

## 31. 制作優先順位

1. **第1段階**: 主人公のデザインを完全に固定する
2. **第2段階**: 主人公の感情表現を作る。パンチやキックより先に、緊張・不安・安心・小さな笑顔・決意・優しさを完成させる
3. **第3段階**: サイト用15秒映像を完成させる
4. **第4段階**: 広告用30秒映像へ展開する
5. **第5段階**: 90秒のブランドムービーを制作する
6. **第6段階**: 3〜5分のショートフィルムを制作する
7. **第7段階**: シリーズ作品、絵本、LINEスタンプ、グッズ、UIZIN連携へ展開する

## 32. 最終判断基準

映像を公開する前に、最後に一つだけ確認します。

> **この作品を、運動が苦手で自信をなくしている子どもが見ても、安心できるか。**

安心できない場合は、どれだけ映像が美しくても修正します。

FLATUP GYMのアニメーションで最も守るべきものは、技術でも流行でもありません。**挑戦する人の気持ち**です。
