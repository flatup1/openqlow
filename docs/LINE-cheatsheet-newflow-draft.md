## 🌅 朝の新しい流れ（④）

毎朝、openQLOW が **今日のおすすめを1つだけ**持ってきて聞きます。

```
openQLOW: おはようございます。今日はこの内容でいいですか？

本文:
（今日の投稿文）

投稿先: all

画像の候補（~/openqlow/media/ から）:
1. kids_practice.jpg
2. gym_front.jpg
3. member_smile.jpg
→ 使う画像の番号を送ってください（例: 画像 1）。画像なしなら「画像なし」。
```

Jin はこの中から選ぶだけ：

| 送る言葉 | どうなる |
|---|---|
| `画像 1` | 1番の画像を使う |
| `画像なし` | 画像なしで進む |
| `OK` | この内容で進む（最後はJinが画面で投稿） |
| `修正 もっと短く` | 本文を直す（→また確認に戻る） |
| `挿入` | フォルダの別の画像を選ぶ |
| （写真/動画を送る） | 送った画像をそのまま使う |
| `NO` | 今日は出さない |

> 候補がない朝は「今日はちょうどいい候補がありません」と出ます。無理に出しません。

## ✏️ 本文を直したい（②）

```
Jin: 修正 セールは6/10までに変更
openQLOW: 下書きを直しました。これでいいですか？
          （新しい本文） → OK / さらに修正 / NO
```

- 直近の下書きが対象です。
- 直しても **すぐには投稿しません**。必ずもう一度 `OK` を聞きます。

---

## 🖼 フォルダの画像を入れたい（③）

写真・動画は `~/openqlow/media/` に入れておきます。

```
Jin: 挿入
openQLOW: 1. kids.jpg  2. gym.jpg  3. demo.mp4  → 番号を送ってください
Jin: 挿入 2
openQLOW: 添付しました。中身を確認してください → OK / 修正 / NO
```

---

## 📲 LINEから直接 写真・動画を送りたい（⑤）

候補に良いものがない時は、**その場でLINEに写真や動画を送るだけ**でOK。

```
Jin: （写真を送信）
openQLOW: 受け取りました。これを使いますか？ → OK / 修正 / NO
```

---

## 🛡 ずっと変わらない約束

- ❌ openQLOW が勝手に投稿することはありません。最後に Jin が `OK` するまで何も外に出ません。
- ⚠️ 画像は投稿前に必ず Jin が目で確認（お客様・お子さん・女性・会員さんの映り込み防止）。
- ✅ 「投稿しました」と言うのは、本当に投稿できた時だけ。
