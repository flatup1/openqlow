## 0. 一行で言うと

**格闘技を売るホームページではなく、親の「今しかない時間」を描くホームページ。**

## 1. ねらい / ターゲット

- **ねらい**: 訪れた親が数十秒で「ここなら安心して子どもを任せられる」と感じること。売り込みではなく、共感でつながる。
- **ターゲット**: 子どもの成長を大切にしたい親。とくに **4〜12歳** の子がいる家庭。はじめての習い事を検討する層。
- **意思決定者は親**。だから映像は子どもではなく **親の目線と気持ち** を主役にする。

## 2. コアメッセージ(最後に一言だけで回収する)

> 子どもは、言葉よりも **親の背中** を見て育つ。
> ——だから、強いパンチじゃなくて、**強い心**。

締め: **世界一、初心者に優しい格闘技ジム。FLAT UP GYM。**

> 注意: この言葉を映像の途中で説明しない。**映像だけで積み上げ、最後にナレーション一言で"すくい上げる"。** これが本作の心臓。

## 3. 演出テーマ(伏線 → 回収):「親の背中を見て育つ」

本作の背骨。**最初は一切説明せず、映像だけで伏線を積む。** 子どもが親を自然に真似る仕草を、日常のなかに何度もそっと仕込む。見ている親は理由を意識しないまま「あ、うちもそうだ」と感じる。そして最後にナレーション一言で回収する——**見終わった後に、じわっと心に残る**。

伏線の具体(すべてセリフなしで見せる):
- 朝、親が靴を履く。子どもが少し後ろから、その背中を見ている。
- お迎え。親が **目線を合わせてしゃがむ**。子どもが自然と同じようにしゃがむ(真似)。
- ジムで、**親も子も自然に礼をする**(教わったのではなく、見て覚えた所作)。

→ 積み上げた"真似"のすべてが、最後の一言「親の背中を見て育つ」で一気に意味を持つ。

## 4. トーン&マナー

- セリフ・説明は最小。**間・光・音・表情**で感情を伝える(Bible §16「間」/ §17「言葉より行動」)。
- 切なさで終わらせない。**あたたかい希望**へ抜ける。基準は Bible §32「自信をなくした子どもが見ても安心できるか」。
- 怒鳴らない/比べない/急かさない/置いていかない(Bible §3)。

### ★ 演出の心臓:伏線の回収(カット1・2・4 → カット7)
カット1(背中を見る)・カット2(しゃがむを真似る)・カット4(礼を真似る)で積んだ"真似"が、カット7の一言「親の背中を見て育つ」で**一気に意味を持つ**。だから前半で絶対に説明しない。回収は一度だけ。

### ★ 橋渡しの設計(共感 → ジム)
親の時間からジム紹介への移行は**唯一の勝負どころ**。飛躍すると「感動系CMの乗っ取り」に見える。橋は **「強い心」** の一語。子どもが**転んでも自分で立ち上がる**カット5(=既存の世界観「はじめの一歩を、笑わない」)がジムの意味を先に見せているので、カット7〜8で FLAT UP GYM が押し売りでなく **"答え"** として着地する。
→ ここでFLATUPのマスコット(フラッピー/あの女の子キャラ)を「強い心の象徴」としてそっと重ねると、感動パートと既存アニメ資産が1本の線でつながる。

## 6. 制作トラック(正直な現実)

このオープニングは、既存のマスコット素材(girl-power-op のキャラ・ポーズ)**だけでは作れない**。親・子・保育園・手のクローズアップという新しいアートが必要。おすすめの二層構成:

- **前半(カット1〜6)**: 親子の実写風 or 新規イラスト/3Dで、日常と手の成長を描く。**新規アセットが必要**。
- **後半(カット7〜8)**: 「強い心」の象徴として **既存マスコット(フラッピー)** が登場し締める。**girl-power-op の資産が活きる**(entrance / help / sunrise / cta / rest 等のシーン型、CTAボタン=`lin.ee/cTSDajPz`)。

これにより「感動 → ジムらしさ → 予約」が滑らかにつながり、既存実装(girl-power-op / animation-studio)も無駄にならない。

## 7. 音・音楽(Bible §19-20 準拠)

- 導入は小さなピアノ。生活音・子どもの声を優先。
- カット5で一瞬の静寂(間)。カット6で温かい旋律がひらく。
- テーマ曲は**最後**(感動→行動)でのみ完成させる。ファンファーレにしない。

## 8. サイト実装への接続

- この映像はトップのファーストビュー。音声は自動再生しない(ユーザー操作で再生)。
- 映像後に自然に接続: 500円体験 / 初心者・女性・キッズ案内 / よくある不安 / LINE。
- 既存の埋め込み方式: `girl-power-op/index.html?loop=1`(後半マスコットパートのループ用)。

## 9. KPI(作品でありマーケ資産)

再生率 → 完視聴 → LINE追加 → 500円体験 → 入会 → 口コミ。
成否は「美しさ」ではなく **「怖くなさそう/自分にもできそう/ここなら任せられる」と感じた割合** で測る。
