# 📖 openQLOW LINE 使い方ガイド（中学生でもわかる版）

> このファイルは Jin が毎日 LINE で openQLOW (MOLTBOT) を使うときの早見表です。
> 困ったときはここを見れば思い出せます。

---

## 🌟 まずこれだけ覚えればOK（毎日使う3つ）

| 送る言葉 | 何が起きる？ |
|---|---|
| **`日報`** | 1問ずつ8個の質問が来る → 1タップで答える → 保存 + 今日やること3つを提案 |
| **`月報`** | 今月の日報を全部、日付順で見せてくれる |
| **`ok`** | 直前の投稿候補を承認し、投稿準備キューまで進める（外部投稿はまだしない） |

**この3つで90%の用事はカバーできます。**

---

## 🌅 ① 朝の日報（毎朝5分・新しい標準）

### Jin が送る
```
日報
```
※ `おはよう` `朝` `daily` でも同じ

### openQLOW が即返す（1問だけ）
```
OPENQLOW（記憶係）：おはようございます。1問ずつ聞きます。

1/8: 昨日、体験に来た人はいましたか？（なければ「なし」）

（途中でやめたい時は「中止」/ まとめて送りたい時は「日報まとめ」）
```

### Jin → 1タップで返信
```
なし
```
*→ 即座に Q2 が届く*

```
2/8: 入会した人はいましたか？
```

### 〜8問繰り返す〜

### 最後に保存＋ToDo3つが届く
```
OPENQLOW（記憶係）：保存しました📝
日付: 2026-06-04
（新規ファイル）

今日やることはこの3つです：
1. 今日のタスク: 看板撮影
2. 入会検討中の方へ声かけ: 佐藤さん料金
3. 退会リスク確認: せとさん

必要なら、このままLINE下書きも作れます。
```

### 💡 なし のいろんな書き方が全部「なし」になる
| 入力 | どう扱われる |
|---|---|
| なし / ナシ / ない / 無 / 無し | ✅ なし |
| なひ（フリック入力ミス） | ✅ なし（自動救済） |
| n / N / no / NO / none / N/A | ✅ なし |
| - / — / ー | ✅ なし |
| 特になし | ✅ なし |
| ない人 / 無理 / なしです | ❌ そのまま実値として残る |

## 🚀 ② まとめモード（テンプレで一気に送りたい時）

8問やり取りが面倒な日は、こっちで1メッセージで完結。

### Jin が送る（スペースなしOK）
```
日報まとめ
```
※ `日報 まとめ`（スペース有り）/ `おはようまとめ` / `日報bulk` / `日報テンプレ` でも同じ

### openQLOW が返す
8項目テンプレ：
```
1. 昨日の体験：
2. 入会：
3. 入会迷ってる人：
4. 返信・フォローが必要な人：
5. 口コミ頼めそうな人：
6. 休みがち・退会しそうな人：
7. 気になる会員：
8. 今日の最優先タスク：
```

### Jin がコピペで埋めて送信（番号さえ守ればOK）

## 📸 ④ SNS投稿（写真・動画つき）

### 朝の1候補確認
日報保存後に `投稿` と送ると、本文1案だけを返します。

```
投稿
```

返ってくるもの：
- 本文1案
- 投稿先（Threads / Googleビジネスプロフィール / LINE VOOM）
- 画像候補（`OPENQLOW_MEDIA_DIR` から最大5件）

次の操作：
| 送る言葉 | 結果 |
|---|---|
| `画像 1` | 1番目の画像を添付。目視確認に戻る |
| `画像なし` | 画像なしで進める |
| `修正 新しい本文` | 本文を書き換え、安全チェック後に再確認 |
| `挿入` | フォルダ内の画像/動画候補を一覧表示 |
| LINEに画像/動画を直接送る | 直近の承認待ち下書きに添付。目視確認に戻る |
| `OK` | 投稿準備キューまで進める。外部投稿はしない |
| `NO FG-...` | 取り下げ |

画像・動画は、必ずJinが映り込みを目視確認してから進めます。

### ステップ1: ファイルを準備
画像・動画を**ここに置いてください**（未設定時の標準）：
```
~/openqlow/media/
```
別フォルダを使う場合は `OPENQLOW_MEDIA_DIR` を設定します。

### ステップ2: LINE で投稿依頼
```
/投稿
本文: 今日のキッズ練習風景
ファイル: /Users/jin/Desktop/openqlow-posts/ready/videos/sunny.mp4
投稿先: all
```

投稿先の選択肢：
- `all` … Threads / LINE VOOM / Google ビジネス全部
- `threads` … Threadsだけ
- `line_voom` … LINE VOOMだけ
- `google_business` … Googleビジネスだけ

### ステップ3: openQLOW が候補を返す
```
画像/動画つき投稿候補を作りました。
ID: FG-20260604-901
投稿先: threads, line_voom, google_business

この内容で投稿するなら: ok
下書き保存だけ: OK FG-20260604-901
投稿準備まで: OK FG-20260604-901 all
```

### ステップ4: 承認
| 送る言葉 | 結果 |
|---|---|
| `ok` | 直前候補を承認（投稿準備キューまで。外部投稿はまだしない） |
| `OK FG-20260604-901` | 下書き保存だけ |
| `OK FG-20260604-901 all` | 投稿準備キューに入れる |
| `やめる FG-20260604-901` | 却下 |
| `修正 新しい本文` | 直近の承認待ち下書きを書き換えて再確認 |
| `修正 FG-20260604-901: 新しい本文` | 指定IDの本文を書き換えて再確認 |

### ステップ5: Mac側で最終投稿
```bash
cd "/Users/jin/Desktop/OPENQLOW HelMES/openqlow"
node scripts/run-vps-browser-post.mjs FG-20260604-901
```
→ Chrome が開く → 本文クリップボード → 動画添付 → 最終投稿ボタンはJinが確認して押す

## ⚠ ⑦ 困った時

| 症状 | 対処 |
|---|---|
| 反応がない | スマホ入力で全角コマンドかも → `日報` だけ送って確認 |
| 質問の途中で間違えた | `中止` と送ってやり直し |
| 保存されたか不安 | `月報` で今日の分が見えれば OK |
| 投稿IDを忘れた | `ok` だけで直近候補が投稿準備キューまで進む |
| `EACCES` エラー | VPS再起動：`ssh -i ~/.ssh/openqlow_vps root@162.43.41.182 'systemctl restart openqlow-webhook.service'` |
| LINE で返事こない | webhook プロセスが落ちてるかも。VPS再起動で復活 |

## 🎯 ⑧ 一番大事な原則

> **「openQLOWが勝手に投稿することはない。最後にJinが `ok` と言うまで何も外に出ない。」**

- ❌ 自動投稿しない
- ❌ 自動でお客さんにLINEしない
- ❌ 自動で入会・退会・返金を決めない
- ✅ 全ての公開操作は人間承認後
- ✅ ファイル削除も承認後

```
07:05  Jin:      ok

```

**3分で1日が始まる。**
