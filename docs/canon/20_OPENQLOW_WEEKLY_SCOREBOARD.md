# 20 openQLOW週次スコアボード（WEEKLY SCOREBOARD）

作成: 2026-07-19 ／ 担当: openQLOW（集計・警戒サイン・提案まで）。数字の記録元はJINの実務。
売上導線: **体験予約 → 入会 → 継続 → 口コミ → 紹介**

前提（重要）:
- openQLOWはお客様へ自動送信・自動返信しない。お客様のメール自動受信も前提にしない。
- ここに書く数字は、JINが把握できる範囲の**手集計でよい**。CRM（`src/crm/`）に入っている問い合わせ記録は補助として使う。
- 「JINの承認が必要な操作」列の操作は、openQLOWは**下書き作成まで**。

## 記録テンプレ（毎週1行増やす）

```
| 週 | 問い合わせ | 体験予約 | 体験実施 | 入会 | 退会・休会 | 在籍数 | 口コミ新規 | 紹介経由 | メモ |
```

---

## 段階1: 体験予約

| 項目 | 内容 |
| --- | --- |
| 最低限記録する数字 | ①新規問い合わせ数（LINE）②体験予約数 ③体験実施数（無断キャンセル数も） |
| 数字の意味 | 導線の入口の太さ。①→②は返信の質、②→③は前日リマインドの質 |
| 警戒する状態 | 問い合わせ0件が2週連続／予約→実施が半分以下（無断キャンセル多発） |
| 次に確認すること | 返信までの時間・返信内容が正本FAQどおりか・リマインドを送ったか |
| 推奨アクション | AIKA返信下書きの改善／前日リマインド文の下書き／曜日枠の案内順の見直し |
| JINの承認が必要な操作 | リマインド送信・体験日時の確定・LINE配信 |

## 段階2: 入会

| 項目 | 内容 |
| --- | --- |
| 最低限記録する数字 | ①体験実施数 ②入会数 ③体験後1週間以内の追客連絡数 |
| 数字の意味 | 体験→入会率はジムの体験価値そのもの。追客は1回だけ・優しく |
| 警戒する状態 | 体験3件以上で入会0が2週連続／追客をしないまま1週間経過 |
| 推奨アクション | 体験当日の声かけ内容の振り返り／追客文（T14）の下書き／入会手続きの案内資料整備 |
| 次に確認すること | 体験者の不安（料金・続けられるか・雰囲気）が解消されたか。体験前チェックの記録 |
| JINの承認が必要な操作 | 追客文の送信・入会手続き・料金や割引の判断 |

## 段階3: 継続

| 項目 | 内容 |
| --- | --- |
| 最低限記録する数字 | ①在籍数 ②当月の退会・休会申請数 ③2週間以上来ていない会員数（分かる範囲） |
| 数字の意味 | 月々の売上の土台。退会は「翌月末」ルールなので、申請数が翌々月の売上を予告する |
| 警戒する状態 | 退会申請が月2件以上／来なくなった会員が増える／同じ理由の退会が続く |
| 次に確認すること | 退会理由の傾向・クラス満足度・カードキー返却と会費ペイ停止の処理漏れ |
| 推奨アクション | 声かけ対象リストの作成／退会理由の分類メモ／レッスン内容の改善提案 |
| JINの承認が必要な操作 | 退会・休会の成立処理・会費ペイ停止・引き止めの連絡 |

## 段階4: 口コミ

| 項目 | 内容 |
| --- | --- |
| 最低限記録する数字 | ①Google口コミ件数と平均点 ②今週の新規口コミ数 ③口コミ依頼の下書き作成数 |
| 数字の意味 | 体験前の人が最初に見る「安心材料」。件数より継続的に増えることが大事 |
| 警戒する状態 | 1ヶ月間新規0件／低評価がついたのに気づいていない |
| 次に確認すること | 依頼のタイミング（体験直後・上達の節目）を逃していないか。低評価の内容 |
| 推奨アクション | 口コミ依頼文の下書き（体験後・UIZIN後）／低評価への返信下書き（謝意+改善） |
| JINの承認が必要な操作 | 口コミ依頼の送信・口コミへの返信投稿 |

## 段階5: 紹介

| 項目 | 内容 |
| --- | --- |
| 最低限記録する数字 | ①紹介経由の問い合わせ・体験数 ②成長共有LINE（写真つき）の送信数 ③イベント（オープンデー・UIZIN）の見学者数 |
| 数字の意味 | 「言わずに発生させる」仕組み（REFERRAL_PLAYBOOK）が回っているかの結果指標 |
| 警戒する状態 | 成長共有の送信が0の週が続く／イベント告知をしないまま月末 |
| 次に確認すること | レッスン中の写真撮影習慣・QRカードの設置と残枚数・次回イベント日程の確定 |
| 推奨アクション | 成長共有文（テンプレA/B/C）の下書き／イベント告知文の下書き／紹介特典（バンテージ）の案内漏れ確認 |
| JINの承認が必要な操作 | 保護者への写真送信・LINE一斉配信・イベント日程の確定・特典の進呈 |

## 全体の見方

- 数字は**左の段階から順に**見る。入口（問い合わせ）が細いのに出口（紹介）を触らない。
- 1週間の変動で慌てない。**2週連続の警戒状態**で初めて `22_OPENQLOW_NEXT_ACTION_RULES.md` の判断表へ。
- 集計がしんどい週は「問い合わせ・体験実施・入会・退会申請」の4つだけでもよい。続けることが最優先。
