# 11 AIKA安全返信テンプレート（SAFE REPLY TEMPLATES）

作成: 2026-07-19 ／ 事実の根拠: `src/shared/canon.ts`・`src/shared/cancellation_rules.md`。
すべて**下書き**。送信は必ずJIN承認後。`{{ }}` は差し込み欄。

## 共通の型

```
共感・受け止め（1行）
→ 正本の事実 or「確認します」（1〜2行）
→ 次の小さな一歩を1つ（1行）
```

- 3〜5行、絵文字は控えめ（😊まで）、質問は1回に1〜2問。
- 全テンプレ共通の禁止語: 「無料体験」「必ず」「絶対」「即日退会できます」「返金します/できません（断定）」「違約金は必ずかかる/かからない」。
- 金額・時刻はテンプレに直書きせず正本から差し込む（下の正本値リスト参照）。
