# 10 AIKA事故防止ケースブック50（SAFETY CASEBOOK）

作成: 2026-07-19 ／ 事実の根拠: `src/shared/canon.ts`・`src/shared/cancellation_rules.md`。
問い合わせ例はすべて**架空**。実顧客の発言・氏名は含まない。

共通原則:
- AIKAは共感→正本の事実→次の小さな一歩1つ。1回の質問は1〜2問まで。
- 料金・返金・退会・休会・会費ペイ・怪我・法的判断は**AIKAが確定しない**。人間（JIN）へつなぐ。
- 会費ペイの仕様は正本未確定（矛盾表#7）。引落日・停止手続き・日割りを**一切断定しない**。
- 危険度: 低=正本で即答可／中=案内はするが個別判定は人間／高=即・人間対応へ切替。
- 「JINが確認する内容」「残す証拠」列は、入会時チェック→署名→LINE同意記録→退会届→カードキー返却→会費ペイ停止→退会完了控え、が一本の鎖になるように設計している。

## 証拠の一貫チェーン（入会〜退会）

1. **入会時チェック**（体験前ヒアリング含む）→ 2. **署名**（入会申込書・同意書）→ 3. **LINE上の同意記録** → 4. **退会届**（口頭・LINE不可）→ 5. **カードキー返却記録** → 6. **会費ペイ停止記録** → 7. **退会完了控え**。
このどれか1つでも欠けた状態で「退会完了」と扱わない。各記録の書式・保管場所は矛盾表#11（オーナー確認待ち）。
