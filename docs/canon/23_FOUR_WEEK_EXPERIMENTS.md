# 23 4週間改善実験（FOUR WEEK EXPERIMENTS）

作成: 2026-07-19 ／ 費用と作業量が**少ない順**に週1実験。
評価は「広告や投稿の本数」ではなく、**売上導線のどの段階の数字が動いたか**で行う。
同時に動かす実験は1つ。前の週の実験は止めずに習慣化してよい（積み上げ式）。

共通ルール:
- すべての送信はJIN承認後。openQLOWは下書きと計測まで。
- 判定は「やったか」と「対象段階の数字」の2つだけ記録する（`20`のメモ列）。
- 4週すべての開始前提: 週次スコアボードの記録が回っていること（週1行）。

---

## 第1週: 返信の型の徹底（費用0円・作業ほぼ0）

- **やること**: 今週の全問い合わせ返信を「共感→正本の事実→次の一歩1つ」の型（`11_AIKA_SAFE_REPLY_TEMPLATES.md`）に統一する。質問は1回1〜2問まで。
- **対象段階**: 段階1（問い合わせ→体験予約）。
- **計測**: 問い合わせ数、そのうち体験予約まで進んだ数、返信までの時間。
- **成功の目安**: 問い合わせからの予約転換が先週より1件でも増える。または返信時間が縮む。
- **費用/作業**: 0円／AIKA下書き+JIN承認のみ（普段の返信作業と同じ）。

## 第2週: 前日リマインドの習慣化（費用0円・1件1分）

- **やること**: 体験予約の前日に、リマインド下書き（持ち物・場所・「お気をつけてお越しください」）を必ず用意し、JINが承認送信する。
- **対象段階**: 段階1後半（予約→実施）。無断キャンセルの削減。
- **計測**: 予約数、実施数、無断キャンセル数。
- **成功の目安**: 予約→実施率が上がる（目安: 8割以上）。
- **費用/作業**: 0円／1件あたり1分。

## 第3週: 成長共有LINE週3件（費用0円・写真撮影の習慣）

- **やること**: レッスン中に写真3〜5枚撮り、保護者・会員向けの成長共有文（REFERRAL_PLAYBOOKテンプレA/B）を週3件送る（JIN承認後）。顔出しNGの確認を必ず先に行う。
- **対象段階**: 段階3（継続）と段階5（紹介）の種まき。
- **計測**: 送信数、返信・反応数、紹介・見学の問い合わせ数。
- **成功の目安**: 保護者からの反応が返ってくる。2〜4週遅れで紹介問い合わせが出れば大成功。
- **費用/作業**: 0円／撮影+承認で週15分。

## 第4週: 口コミ依頼とQRカード設置（費用約1,000円）

- **やること**: ①今月の体験者・節目の会員に口コミ依頼文を下書き→JIN承認で依頼（1人1回だけ）。②QRカード（名刺サイズ・約1,000円/100枚）を発注しジム入口へ設置。
- **対象段階**: 段階4（口コミ）と段階5（紹介）。
- **計測**: 新規口コミ件数、QRカードの持ち帰り枚数（設置枚数−残枚数）、紹介経由問い合わせ数。
- **成功の目安**: 新規口コミ1件以上。カードが減り始める。
- **費用/作業**: 約1,000円／依頼下書きは AIKA、発注・設置はJINで計30分。

## 4週後の振り返り方

1. スコアボードの4週分の行を並べ、段階1→5のどこが動いたかを見る。
2. 動いた実験は**習慣（SOP）へ昇格**させ、`21_OPENQLOW_WEEKLY_SOP.md` に追記提案する。
3. 動かなかった実験は、やり方を変えるか一旦止める（罪悪感なく捨てる）。
4. 次の4週の実験候補（イベント開催・体験2回目導線など、費用と作業が一段大きいもの）はこの結果を見てから設計する。
