# 紹介を「言わずに」発生させるプレイブック

作成日：2026-06-08
位置づけ：戦略書 `STRATEGY_2026_AB_HYBRID.md` §4 の実装詳細
対象：JIN

---

## 0. なぜこのプレイブックか

「友達連れてきてね」「紹介してください」が JINにとって苦手。
FLATUP の優しいブランドにも合わない。

→ JINが**1回も依頼を口にせず**、紹介を発生させる仕組みをまとめた。

---

## 🚫 JIN が言わない言葉リスト

```
× 友達連れてきてね
× 紹介してください
× お友達紹介キャンペーン参加して
× 家族・友達誘って来てください
× 紹介者募集中
```

---

## ✅ JIN が「言ってOK」な言葉リスト

### イベント告知系
- 「来月、UIZINやります」
- 「7/20はオープンデーです」
- 「見学OKですよ」
- 「気になる方いらしてください」

### 子どもの成長共有系
- 「○○ちゃん今日こんなことできましたよ」（写真つき）
- 「今日のレッスン、すごく良い顔してましたね」
- 「○○くん、3ヶ月でこんなに変わりましたね」

### 受身の言葉
- 「気になる方いたら教えてください」
- 「もしよかったら」
- 「お時間あれば」

## 1. 写真撮影習慣（毎レッスン）

### ルール

```
タイミング：レッスン中・終わり
枚数：1人につき3〜5枚
内容：
  ✅ 笑顔 / 真剣な顔 / フォーム決まった瞬間
  ✅ ハイタッチ / 仲間とのやりとり
  ✅ 成長の証（できなかったことができた瞬間）
  ❌ 顔出しNG家庭の子（事前に名簿で確認）
```

### 送信テンプレ A：通常レッスン後

```
☀ 本日のレッスン記録

○○ちゃん、今日のミット打ちすごく集中していました！
1ヶ月前と比べてフォームも安定してきています。

（写真3枚）

ご家庭でも頑張りを聞いてあげてくださいね。

FLATUP GYM
```

### 送信テンプレ B：新しいことができた日（特別バージョン）

```
☀ 今日の成長記録

○○くん、初めて回し蹴りができました！
3週間前は「むずかしい」と言っていたのに、自分で考えて
動けるようになりました。

（写真3枚 + 動画1本）

「敵は相手じゃない、敵は自分」を、
今日の○○くんが体現してくれました。

おめでとうございます。

FLATUP GYM
```

### 送信テンプレ C：UIZIN後

```
☀ UIZIN参加おめでとうございます

○○ちゃん、初陣を堂々と乗り越えました。
勝ち負けじゃなく、自分の弱さに向き合えたこと、
それが何より誇らしいです。

（集合写真 + ○○ちゃんの瞬間3枚）

このがんばり、ご家族みんなで分かち合ってください。

FLATUP GYM
```

→ 親のSNS拡散が「自然な紹介」を生む。JINは何もお願いしてない。

## 4. シーン別の言い方（実演）

### シーン1：レッスン終わり、保護者と話す時

```
× 「お友達も習いたい子いたら連れてきてください」

○ 「○○ちゃん今日めちゃくちゃ集中してましたね！
    （写真見せる）
    今度UIZINでこういうのやるんですよ」

→ JIN は子どもの話をしているだけ
→ 親が「他にも誘っていい？」と聞いてくる
```

### シーン2：LINE配信

```
× 「紹介キャンペーン実施中！友達を誘って！」

○ 「☀ 7/20 UIZIN開催のお知らせ
    会員さんのご家族・お友達も見学OKです
    詳しくは @jfl0054o」

→ JIN は配信を「ok」承認するだけ
→ 文面は openQLOW が作る
```

### シーン3：紹介QRカード

```
× JINが手渡しで「これ友達に渡してね」

○ ジム入口に置いておくだけ
    会員が勝手に持っていく

→ JIN は1回も口を開かない
```

## 5. 紹介発生のメカニズム

```
[JIN行動]                [起きること]
─────────────────────────────────────
レッスン中に写真撮る     → 親に LINE 送付
   ↓                       ↓
                          親が嬉しくて SNS 投稿
   ↓                       ↓
オープンデー告知          友達が「ここ気になる」
   ↓                       ↓
ただ「見学OK」と言う      友達が見学に来る
   ↓                       ↓
                          雰囲気気に入る
   ↓                       ↓
                          体験予約 → 入会
   ↓                       ↓
親にアンバサダー認定       「紹介ありがとう」
   ↓                       ↓
                          次の友達も紹介
```

→ どこにも「友達連れてきてね」が無い。

## 7. JIN に求める「言葉スキル」

```
1. 「ありがとう」が言える ✅（誰でも言える）
2. 子どもの成長を見つけて伝える ✅（JIN得意・FLATUPの核）
3. イベントを「やります」と告知する ✅（事実）
4. 質問されたら答える ✅（受身でOK）

これだけ。営業話法ゼロ。
```
