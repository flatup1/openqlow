# テンプレート：問い合わせ返信

> 料金・日時は `canon/pricing_and_schedule.md` を確認して埋める。空欄は推測しない。
> 3案（丁寧版 / 標準版 / 短文版）で出す。

## 標準版（例：初心者の体験問い合わせ）

```
{{お名前}}さん、お問い合わせありがとうございます。FLATUP GYM です。

はじめての方も大丈夫です。初回体験は500円・約40分（説明込み）で、動きやすい服・タオル・水があればOKです（軍手でも大丈夫です）。

ご希望の曜日・時間帯はありますか？平日で候補をご案内しますね。
（日曜・祝日はお休みのため、平日でのご案内になります）
```

## 丁寧版

- 冒頭に感謝＋不安への配慮を厚めに。
- 料金・持ち物を箇条書きで整理。
- 最後に「次に答えてほしいこと」を1つ明記。

## 短文版（LINE 3〜5行）

- 感謝 → 体験の要点1つ → 質問1つ。

## 確認事項（返信に含める用）

- 未確定の日時 / 料金の最新値 / 退会・休会・違約金 → 「担当より正式にご案内します」
