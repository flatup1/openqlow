# テンプレート：SNS投稿（媒体別）

> コア原稿を1つ作り、媒体別に冒頭・文章量・締めを変える。料金・日時は正本参照。

## Instagram（キャプション）

- 冒頭1行で共感（例：「運動が苦手でも大丈夫です」）
- 本文3〜5行、改行広め
- 締め：体験の一言案内＋保存/プロフィールリンク誘導
- ハッシュタグは末尾にまとめる（成田 / 格闘技 / 初心者 など）

## X（ポスト）

- 140字前後、1メッセージ1メッセージ
- 冒頭に結論、リンクは末尾

## Threads

- 会話調でやわらかく。1〜2文＋問いかけ。

## LINE VOOM

- 既存フォロワー向け。体験・イベントの実用情報を中心に。

## Googleビジネスプロフィール

- 事実ベース（営業時間・体験・イベント）。誇大表現なし。

## HP記事

- 見出し＋本文。検索意図（初心者 / 女性 / キッズ）に答える構成。

## 共通の禁止

- 医療効果の断定 / 実績・数字の捏造 / 恐怖をあおる表現
- 使用権のない画像・動画
