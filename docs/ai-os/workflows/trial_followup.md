# ワークフロー：体験前後のフォロー

対応する Skill：`flatup-trial-followup`

## タイミング別

1. **体験前日**：リマインド（日時・持ち物・場所）。テンプレ `templates/trial_reminder.md`。
2. **体験当日**：来館前の一言案内。
3. **体験後（当日〜翌日）**：お礼＋感想を聞く。テンプレ `templates/trial_followup.md`。
4. **入会案内**：圧をかけず、次の一歩を明確に。
5. **口コミ依頼**：入会後・満足確認後のみ。

## 原則

- 体験者に圧力をかけない（`canon/brand_voice.md`）。
- 次の行動（返信すべきこと / 予約 / 見学）を1つに絞って明確化。
- 料金・日時は `canon/pricing_and_schedule.md` を確認。
- 送信は人間。AI は下書きまで。
