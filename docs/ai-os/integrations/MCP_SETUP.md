# MCP・外部連携

これは接続候補の台帳であり、接続済みの宣言ではありません。2026-07-18時点のこのCodex環境ではGitHub、Gmail、Google Drive/Sheets、ブラウザ系の機能が見えますが、実際の権限は利用時に再確認します。

| 候補 | 読み取り開始 | 書き込み | 個人情報 | 承認が必要な操作 | 未接続時 |
|---|---|---|---|---|---|
| GitHub | 可 | commit/push/PR等 | 場合あり | すべての重要変更 | ローカル差分だけ作る |
| Gmail | 検索・要約 | 送信・転送 | あり | 送信前 | 下書き文を出す |
| Google Drive/Sheets | 閲覧 | 更新・共有 | あり | 更新・共有前 | ローカル表を作る |
| Google Calendar | 接続確認後 | 予定登録・変更 | あり | 登録・変更前 | 候補日時だけ出す |
| ブラウザ | 閲覧中心 | フォーム・公開 | 場合あり | 送信・公開前 | 手順を案内する |
| LINE公式 | ログ確認のみから | push/reply | あり | 送信前 | 下書きだけ作る |
| 会費ペイ | 原則確認から | 契約・課金 | 高い | 常に人間判断 | 手順と確認事項だけ |
| Instagram/GBP | 公開状態確認 | 投稿・広告 | 場合あり | 公開前 | 投稿下書きを作る |

認証情報は環境変数または各サービスの認可画面で管理し、ファイルへ直書きしません。存在しないMCPサーバーや架空のAPI接続は作りません。
