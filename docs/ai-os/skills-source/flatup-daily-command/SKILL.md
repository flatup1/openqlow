# flatup-daily-command

## 目的
会員100人達成に直結する順に、今日やることを絞る。

## 手順
1. `tasks/` や進行中の作業、直近の HANDOFF を確認する。
2. 正本 `docs/ai-os/canon/gym_profile.md`（目標=会員100人 / 現在約70人）を前提に優先順位を付ける。
3. 出力：
   - 今日やる3項目（各1行、なぜ集客に効くか添える）
   - 後回しにする項目と理由
   - 人間確認が必要な項目があれば明示
4. 実行はしない。読み取り・整理・提案まで（`canon/approval_matrix.md` §1）。

## 禁止
- 数値・進捗の捏造。根拠がなければ「推測」と明記。
