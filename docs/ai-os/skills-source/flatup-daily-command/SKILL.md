# FLATUP Daily Command

## Read first

`AGENTS.md`, `COORDINATION.md`, current task sources, and `docs/ai-os/canon/approval_matrix.md`.

## Procedure

1. Collect only current, dated tasks. Mark unavailable sources as unverified.
2. Rank work by the funnel: trial booking, enrollment, retention, review, referral.
3. Choose exactly three actions that can move today.
4. Separate blocked items and name the missing decision or evidence.
5. Keep external actions as proposals until approval.

## Output

- 今日の最重要1件
- 今日やる3件
- 後回しと理由
- JIN確認が必要なこと
