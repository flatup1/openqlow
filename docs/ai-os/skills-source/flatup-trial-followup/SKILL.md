# FLATUP Trial Follow-up

## Procedure

1. Confirm the stage: before, same day, immediately after, next day, enrolled, or review-ready.
2. Verify the booking and facts before naming a date or time.
3. Use `docs/ai-os/workflows/trial_followup.md` and the matching template.
4. Mention one real positive detail only when evidence exists.
5. Make the next action clear without pressure.
6. Stop at a draft and request approval before sending.

If the person declined, complained, was injured, or has a payment issue, switch to human escalation.
