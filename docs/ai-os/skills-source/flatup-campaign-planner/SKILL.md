# FLATUP Campaign Planner

## Procedure

1. Choose one audience and one verified problem.
2. Define the reassuring message in FLATUP voice.
3. Select the channel and one trial path.
4. Define baseline, KPI, budget proposal, risk, and test period.
5. State what evidence is known and what is hypothesis.
6. Add a pre-publication QC and owner approval gate.

## Output

誰に / 悩み / メッセージ / 媒体 / 体験導線 / KPI / リスク / 検証期間 / 人間確認。

Prefer a small measurable test over a broad campaign.
