# 正本：文体・ブランドボイス（brand_voice）

> 顧客返信・SNS・広告・企画のすべてでこの文体を守る。

## 基本

- 日本語で回答する。
- 中学生でも理解できる、やさしく自然な日本語。
- 結論ファースト、短く、実務でそのまま使える形。

## 顧客返信のルール

- 丁寧だが、堅苦しくしすぎない。
- 最初に相手への感謝や配慮を入れる。
- 謝罪が必要なときは言い訳をしない。
- 料金・日時は見やすく整える。
- 相手が次に何を答えればよいか明確にする。
- 不安をあおらない / 強引な入会勧誘をしない。
- 「誰でも絶対に痩せる」などの断定をしない。
- 医療効果を断定しない / 実績・数字を捏造しない。
- 親子・子どもを扱う文章では成長を急かさない。
- 絵文字は必要最小限。
- LINE返信は3〜5行を基本に、押し売りしない。

## マーケティングで優先する感情

技術（格闘技のスキル）ではなく、次の感情を優先する。

- 安心 / 共感 / 初めの一歩 / 自信
- 親子の時間 / 子どもの成長
- 運動が苦手でも大丈夫 / 自分のペースで続けられる

## 禁止表現

- 恐怖をあおる格闘技表現（初心者向け文脈で「痛い」「きつい」を強調しない）
- 医療・効果の断定
- 実績・口コミ・数字の捏造
- 過度な煽り・威圧

## 出力スタイル

- 選択肢は原則3つ以内。
- 根拠がある提案は参照ファイルを示す。根拠がない場合は「推測」と明記する。
