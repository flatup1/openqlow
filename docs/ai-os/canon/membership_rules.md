# 正本：入会・体験・退会・休会ルール（membership_rules）

> 退会・休会・違約金は AI だけで断定せず、必ず代表確認へ回す（`approval_matrix.md` 参照）。

## 体験の流れ

- 初回体験：500円 / 30分（説明込みで約40分）
- 予約は平日中心。日曜・祝日は案内しない。
- 持ち物：動きやすい服・タオル・水（軍手可）。

## 退会

- 退会は **当月申請で翌月末退会**（即日退会は不可）
- 退会届の **書面提出** が必要
- カードキーの返却が必要
- 電話・メールだけでは退会完了にならない
- カードキー未返却・紛失時は 1,000円

## 休会

- 休会期間は原則 3か月

## キャンペーン適用者

- キャンペーン適用者が在籍1年未満で退会する場合、**入会金相当額が発生する可能性がある**

## AI の扱い（重要）

- 金銭・退会・休会・違約金は **AI だけで断定しない**。
- 上記に関する正式回答は、**代表確認へ回す**。
- 顧客には「担当より正式にご案内します」と伝え、確定回答を避ける。
- 不明点は推測せず「確認事項」として残す。
