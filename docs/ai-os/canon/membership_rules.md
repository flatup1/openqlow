# 入会・退会・休会ルール

## 現行ルール

- 退会は当月末までの申請で翌月末退会。即日退会は不可
- 退会届の提出が必要
- カードキー返却が必要。未返却・紛失時は1,000円
- 休会は原則3か月
- キャンペーン適用で入会し1年未満で退会する場合、入会金相当10,000円が発生する可能性がある

## 回答境界

退会・休会・違約金・請求額は、契約内容と正本を確認してもAIだけで正式確定しません。「確認中」として代表へ回します。電話やメールだけで手続き完了と案内しません。

リポジトリ正本は `src/shared/canon.ts` の `cancellation` と `suspension`。詳細な正式回答前には契約書・届出・最新運用を人間が確認します。
