# 正本：入会・体験・退会・休会ルール（membership_rules）

> **退会・休会・違約金の唯一の正本は `src/shared/cancellation_rules.md`（最終更新 2026-07-09）**。
> `src/shared/canon.ts` の `cancellation` / `suspension` はその要約。本ファイルもその写しであり、
> 条件・数値を独自に書き換えない。金銭・退会・休会・違約金は AI が断定せず、人間確認へつなぐ。

## 体験の流れ

- 初回体験：500円 / 2回目以降ビジター 3,000円
- 持ち物：動きやすい服・タオル・水
- 予約は平日中心。日曜・祝日は案内しない（`pricing_and_schedule.md` 参照）。

## 退会（cancellation）

- **当月末日までに申請すると、翌月末で退会**。
- 即日退会は不可。退会届の提出が必要（口頭・LINE のみでは成立しない）。
- 退会月までの月会費は通常どおり発生する。
- 例：7/15 申請 → 8月末退会 ／ 8/1 申請 → 9月末退会。

## 違約金 / 入会金相当（penalty）

- **キャンペーン適用で入会**し、**入会から1年未満で退会**する場合、
  **入会金相当 10,000円が発生する可能性**がある。
- 発生の有無は契約内容・キャンペーン条件による。**AI は断定せず、人間（スタッフ／Jin）が契約を確認**して判定する。

## 休会（suspension）

- 原則 3ヶ月まで。詳細な手続き・再開条件はスタッフに確認する。

## AI の対応方針（重要）

- 手続きや金額をその場で確定・約束しない（退会処理は Jin の領域）。
- 「即日退会」「違約金は必ずかかる／かからない」などの断定を使わない。
- 一般ルール（申請期限・翌月末退会・退会届が必要・1年未満で入会金相当の可能性）は案内してよいが、
  **個別の該当判定は人間へ渡す**。
- トーンは「世界一優しい格闘技ジム」の温度感。冷たく突き放さず、やさしく次の窓口へ案内する。
