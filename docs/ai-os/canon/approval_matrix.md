# 承認マトリクス

## AIだけで実行可能

- ローカルファイルの読み取り、要約、分析、下書き、誤字確認
- 新規ドキュメント作成、テスト、Git差分確認
- 削除・移動を伴わないファイル健診
- 個人情報を含まない集計と検品

## 実行前に人間承認

- 既存正本、料金、営業時間、規約の変更
- 退会、休会、違約金、返金、請求額の正式回答
- 顧客情報の保存
- commit、push、PR作成、Issue/Project変更、デプロイ
- メール、LINE、SNS、カレンダー、請求書の送信・登録・公開
- 外部サービスへの書き込み、API実行、課金操作、本番データ変更

## 実行禁止

- 顧客情報や健康情報の公開・無断保存
- APIキー、トークン、パスワードの表示・コミット
- 破壊的な再帰削除、履歴を失うGit操作、強制push
- リポジトリ・プロジェクト・本番データベースの削除
- 承認なしの自動投稿・自動送信・金銭処理
- 実績、口コミ、予約枠、料金、健康効果の捏造

## 承認時の表示

```text
確認が必要です。
対象: 対象を具体的に記載
内容: 実行すること
リスク: 誤送信・削除・本番反映・料金判断など
この内容で進めてもよろしいでしょうか？
```
