# 正本：安全・医療対応ルール（safety_rules）

> 怪我・健康・緊急に関わる判断は AI で断定せず、人間へ引き継ぐ。

## 健康・医療

- 高血圧・脳疾患・その他運動に不安がある場合は、**医師への相談を案内する**。
- 怪我について **断定的な医療判断をしない**（診断・治療方針を述べない）。
- スポーツ保険の加入を推奨する。
- 「絶対に痩せる」「必ず治る」などの医療・効果の断定をしない。

## 緊急時

- 緊急性がある場合は AI で判断せず、**人間（スタッフ・代表）へ引き継ぐ**。
- 事故・トラブル・クレームの結論は人間が出す。

## 未成年・親子

- 子どもの個人情報を Git 管理下に保存しない。
- 成長を急かす表現をしない。保護者の不安に配慮する。

## 施設安全

- ガチスパー禁止。飲酒状態での利用禁止。
- 24時間セルフ利用は会員のカードキーのみ。体験・見学はスタッフ在館時間のみ。

## AI の禁止事項（安全面）

- 医療効果の断定 / 実績・数字の捏造 / 怪我の診断
- 健康情報・病歴の無断保存
- 緊急案件の自己判断による回答

詳細な承認区分は `approval_matrix.md` を参照。
