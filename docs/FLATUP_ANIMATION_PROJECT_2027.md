## プロジェクトビジョン

FLATUP GYMは格闘技ジムではありません。私たちが届けたいのは **「人生を変える最初の一歩」** です。

その価値を文章ではなく、**映像・音・感情・キャラクター・物語**で伝えるために、FLATUP GYMだけのオリジナルアニメーションIP（知的財産）を構築します。

目標は単なる広告ではありません。サイトを開いた瞬間、「ここなら自分でも大丈夫かもしれない」と思ってもらえる体験を作ります。

## プロジェクト理念

世界中には技術的に優れたアニメがあります。しかし私たちが目指すのは **初心者の不安を消すアニメーション** です。

主人公は最初から強くありません。失敗します。転びます。怖がります。
でも、誰も怒りません。誰も笑いません。それがFLATUP GYMです。

## ミッション

映像を見た人が「強そう」ではなく **「行ってみたい」** と思う作品を作る。

## 世界観

舞台は夜のFLATUP GYM。昼間は人間たちが練習しています。

夜になると、グローブ・バンテージ・サンドバッグ・トロフィー・キャラクター達が静かに命を持ち始めます。彼らは誰にも知られず、昼間に挑戦する人達を応援しています。

## テーマ

作品全体で描くテーマは「勇気」ではありません。もっと小さなもの——**「最初の一歩」**。
たった一歩だけ。その一歩を肯定する世界。

## ストーリー

主人公はFLATUP GYMの小さなマスコット。まだパンチも打てません。グローブは大きすぎます。周りのみんなは上手です。主人公は「ぼくなんか無理だ」と思っています。

しかし誰も急かしません。怒りません。比べません。ただ、隣に立ってくれます。

主人公は小さなパンチを一発だけ打ちます。サンドバッグは少しだけ揺れます。でも、主人公の心は大きく動きます。

最後には強いヒーローではなく、**次に来た初心者へ手を差し伸べる存在**になります。

## 感情設計

① 興味 → ② 共感 → ③ 安心 → ④ 挑戦 → ⑤ 感動 → ⑥ 行動

最後は体験予約へ自然につなげます。

## キャラクター設計思想

キャラクターは、かわいいだけでは足りません。見る人が「自分」を重ねられる必要があります。
そのため最初は少し不器用。少し怖がり。少し自信がない。そこから少しだけ成長します。

## 映像品質目標

世界トップクラスの家族向け3Dアニメーション作品から学べるような:

- 感情移入しやすいキャラクター／温かい照明／映画的なカメラワーク
- 豊かな表情／自然な間／小さな仕草で感情が伝わる演技
- 子どもも大人も楽しめるストーリー

既存作品のキャラクターやデザインを再現するのではなく、**感情表現や物語づくりの考え方を参考にしながら、FLATUP GYMだけのオリジナル作品**を制作します（IPルールは ENGINE 正本の絶対ルール参照）。

## 技術コンセプト

PR #51 によって、静止画アニメから本格的なポーズ切替アニメーションへ進化しました。これが **FLATUP Animation Engine 第一世代** です。

- `poses/` に画像を置くだけでシーンごとに自動切替（idle / punch / jump / win…）
- 画像が無ければ `chara.png` へ自動フォールバック
- つまり **1枚から始めて映画品質まで進化できる**

### 次世代構想: 感情演技システム

静止画切替だけでなく、感情の演技を複数ポーズで構成する:

緊張 → 視線が泳ぐ → 呼吸 → まばたき → 拳を握る → 一歩踏み出す → パンチ → 安心した笑顔

## キャラクター制作フロー

Phase1 デザイン研究 → Phase2 FLATUPキャラクター制作 → Phase3 設定資料作成 → Phase4 表情集 → Phase5 ポーズ集 → Phase6 専用LoRA学習 → Phase7 全シーン生成 → Phase8 Animation Engineへ組込み

（Phase1のみ参考LoRA可・内部限定。Phase2以降は完全オリジナル。詳細は ENGINE 正本の3段階ロードマップ）

## 映像演出

重要なのはパンチではありません。**"間"** です。
主人公が少し迷う。少し震える。深呼吸する。小さく笑う。その数秒が感情になります。

## 音響設計

音楽で泣かせるのではありません。環境音で没入させます。
夜のジム・時計・床・呼吸・革の音・朝日・静かなピアノ。最後にだけテーマ音楽が完成します。
