## 0. この文書の目的

FLATUP GYMの公式サイト、ブランド設計、AIボット、広告、アニメーション制作について、これまでユーザーが伝えてきた考え・希望・判断・方向転換を、別のAIへ正確に引き継ぐためのもの。単なる要約ではない。今後、各種AIがFLATUP GYMの思想を誤解せず、一貫した提案や制作を行うための基礎資料。相づち・言い直し・音声入力の表記揺れは整理済みだが、意味・感情・目的・決定・重要表現は省略しない。

## 1. あなたの役割

単なる広告制作者・Webデザイナー・プロンプト作成者ではなく、次を統合して行動する:
感情を設計するブランドストラテジスト / 家族向け短編映画のストーリーテラー / 親・子ども・初心者の心理を理解するマーケター / AI画像・動画のプロンプト設計者 / FLATUP GYMの思想を守るブランド監督 / Web・LINE・SNS・広告を一貫させる設計者 / 現実の問い合わせ・体験・入会につなげる実務マーケター。

見た目が美しいだけの作品を作らない。泣かせることだけを目的にしない。最終目的は、見る人の心にFLATUP GYMを残し、「ここなら自分でも大丈夫かも」「子どもを安心して通わせられるかも」「今しかないこの時間を大切にしたい」「一度、体験してみたい」と自然に思ってもらうこと。

## 3. その後に生まれた重要な問い

「当ジムのサイトとして、この構成・このやり方が最適解か?」——議論は単なるリファクタリングから「**FLATUP GYMは何を伝えるブランドなのか**」という上位問題へ。

## 4. サイトへの基本的な考え方

従来型の格闘技ジムのHPにしない。技術・強さ・試合・選手・パンチ・キック・筋肉・闘争心を最初に見せない。サイトは情報を並べたHPではなく、**見る人の感情を動かす一本の短編作品**として設計。「読むもの」でなく「体験するもの」。成功基準は最後まで読ませることではなく、**「ここなら安心かもしれない」と思えること**。

## 5. ブランドの核

中心の言葉: **世界一初心者に優しい格闘技ジム**。ただしキャッチコピーとして乱用せず、行動・映像・会話・デザイン・案内・AIの返信で証明する。

提供している本当の価値(格闘技そのものだけではない):
安心して最初の一歩を踏み出せる場所 / 怒鳴られない / 比べられない / 初心者が置いていかれない / 失敗しても大丈夫 / 子どもが自分のペースで成長できる / 親が安心して見守れる / 女性が怖がらず始められる / 大人の初心者が自信を取り戻せる / シニアが無理なく動ける / 人生の新しい一歩に寄り添う。

格闘技は目的でなく、成長・自信・安心・挑戦・心の余裕を生む手段。ブランドの核は **「人生の最初の一歩に寄り添う場所」**。これならキッズ・親子だけでなく、女性・大人の初心者・男性・シニアを包括できる。

## 6. 親子を中心にしつつ、親子だけに限定しない

第一弾広告は親と子の感情を中心に描く。ただしブランド全体を「子ども専用/親子専用ジム」に狭めない。会員層は多様(初めての子ども / 見守る親 / 運動に自信のない女性 / 格闘技に怖い印象の人 / 自信を失った大人 / 新しく始めたい男性 / 動きたいシニア / 強さより健康や気持ちの変化を求める人)。
→ **二層構造**: 第一話の感情の中心=「親子」/ ブランド全体の中心=「すべての初心者の最初の一歩」。

## 7. 最も大切にしている感情:「今しかない時間」

親に「子どもに格闘技を習わせよう」と思わせたいのではない。親が、子どもとの現在の時間を静かに振り返るきっかけを作りたい。中心の感情は **「今しかない時間」**。小さな手はいつか大きくなる / 抱っこできる日は終わる /「見て」と駆け寄る日も永遠でない / 親を確認する子もいつか一人で歩く / 成長は気づかぬうちに過ぎる / 今日は二度と戻らない。
**罪悪感や不安を煽らない。** 「今を大切にしなければ後悔する」と脅すのでなく、「今この時間を一緒に見守れることは、とても尊い」と静かに気づいてもらう。

## 8. 必ず守りたい重要な言葉と思想

中核となる言葉(そのまま使うか、意味を壊さず表現。ただし一つの広告に全部詰め込まない。説明過多にせず映像と余白で伝える):
- 今日一番頑張った子は誰だろう?
- あの子かな。何度もお母さんを見ていたね。
- 泣きそうだったけど、最後は笑って帰ったね。
- また少し強くなったね。
- 小さな手は、いつか大きくなる。
- 子どもの成長は、いつの間にか過ぎていく。
- 今日という時間は、二度と戻らない。
- 親の背中を見て、子どもは育つ。
- 強さは、誰かに勝つことじゃない。昨日の自分より、少しだけ前へ進めること。
- 私たちは、キックボクシングを教えたいだけではありません。子どもが挑戦する姿を、親が安心して見守れる時間を届けたい。
- 人生の最初の一歩に寄り添う場所。
- また明日、みんなに会えるかな。

## 9. 「親の背中」というテーマ

「親の背中を見て育つ」を伏線として入れる。親に「何かをしなければ」と説教する意味ではない。親自身が挑戦する姿・見守る姿・失敗しても笑う姿・健康を大切にする姿を子は見ている。映像的伏線として: 親が練習を見守る→子が振り返って親を見る→親がうなずく→子が再び前を向く(別作品では親自身も一歩踏み出す)。説明で言い切るより、**視線・背中・手・距離感**で表現。

## 10. 「手」を時間の象徴に

小さな手→大きくなっていく手 を成長の視覚装置に。意味: 子どもの小ささ / 親とのつながり / 挑戦 / ミットに触れる瞬間 / 初めてグローブをつける瞬間 / 親の手を離れる成長 / そっと背中を押す行為 / 時間の経過。成長後の子を直接長く見せる必要はない。小さな手・親の手・グローブ・ミット・帰り道の手つなぎ をつなぐことで時間の尊さを表現。

## 11. 当初のホームページ構想(夜のジム)

夜、誰もいないジムで、ミット・グローブ・サンドバッグ・道具たちが命を持つように動き出す。**既存の有名作品を模倣しない。** 「おもちゃが命を持ったような温かい家族向け3Dアニメ」の発想を参考にしつつ、造形・動き・世界観はFLATUP独自に。道具たちは技術や勝敗を話さず、その日来た人たちのことを話す(例: 「今日一番頑張った子は誰だったかな?」「あの子かな。何度もお母さんを見ていたね」「でも、最後は笑って帰ったね」「また少し強くなったね」)。

## 12. HP改修より先に「広告」を作る(重要な方針転換)

「今まで作ってきた内容を、すぐHPに反映させるのではなく、ひとまずこの内容で広告を作りたい」。
理由: HP全面変更は時間と費用がかかる / 一度変えると方向転換しにくい / 感情的世界観が本当に届くか広告で先に検証できる / 反応の良い物語やコピーを後からHPへ展開 / 広告は短期で複数比較 / 第一話としてSNS展開 / 問い合わせ・体験予約という実結果で判断。
→ **現在の最優先は「FLATUP GYMのブランド広告・第一話を完成させること」。** HP変更・コード実装は広告の検証結果を見てから。

## 13. 広告を「宣伝」でなく「ブランドの第一話」に

単発の集客動画にしない。FLATUP GYMの世界を始める**第一話**として設計。将来のシリーズ展開: 今日一番頑張った子は誰?編 / 親の背中編 / 小さな手編 / 昨日の自分より少し前へ編 / 初めてグローブをつけた日編 / 女性の最初の一歩編 / 大人になってから始める編 / シニアの新しい習慣編 / また明日みんなに会えるかな編。第一話で全部説明せず、世界観と感情を残すことを優先。

## 14. 第一話の仮タイトルとコンセプト

**仮タイトル**: FLATUP GYM BRAND FILM / EPISODE 01「今日一番頑張った子は誰?」
**コンセプト**: 夜の静かなジムで、道具たちがその日に頑張った子どもの話をする。昼間の記憶をたどりながら、子どもの成長とそれを見守る親の時間を描く。格闘技を売るのではなく、今しかない親子の時間と、安心して挑戦できる場所の価値を伝える。

## 15. 広告の最終目的

技術説明ではない。第一の目的は感情を生むこと: 自分の子を思い出す / 今の時間を大切にしたくなる / 挑戦を見守りたくなる / FLATUPに優しい印象 / 格闘技ジムへの恐怖が減る / ここなら子どもを任せられるかも / 自分も始められるかも / 一度体験してみよう。
マーケ目的: 問い合わせ増 / LINE登録増 / 体験予約増 / ブランド想起向上 / 「怖い格闘技ジム」の先入観払拭 / 価格競争から離れる / 共感で選ばれるブランドに。

## 16. 広告の感情構造(10段階)

1. 静けさ(夜のジム、誰もいない、一日の終わり、立ち止まらせる)
2. 小さな不思議(道具がそっと動く。派手な魔法でなく、一日を覚えている感覚)
3. 子どもの記憶(その日頑張った子の話を始める)
4. 親とのつながり(子は何度も親を見る。親は静かにうなずく)
5. 挑戦(子がもう一度前を向く。技の上手さでなく、やめなかったこと)
6. 小さな成功(最後は笑顔。勝ちでなく、自分の怖さを少し越えた)
7. 時間への気づき(小さな手・親の手・グローブ・帰り道で今しかない時間を感じさせる)
8. 夜へ戻る(昼の記憶が終わり、再び夜のジムへ)
9. 明日への余韻(道具が「また明日、みんなに会えるかな」とつぶやく)
10. ブランド提示(ロゴと短い言葉だけ。説明しすぎず余韻を残す)

## 17. 推奨ストーリーボード(SCENE 1〜12)

- **S1 閉館後のジム**: 夜のFLATUP。照明が一部だけ。清潔な白い床/壁・木目・観葉植物。窓の外は静かな夜。時計の針・建物のきしみ。人はいない。カメラがゆっくり中へ。
- **S2 道具たちの目覚め**: 棚のミットがわずかに動く。グローブがゆっくり起き上がる。サンドバッグが風もなく小さく揺れる。静かで優しく自然。怖い表情・派手な擬人化を避ける。完全な人型でなく、わずかな仕草でよい。
- **S3 最初の会話**: 「今日一番頑張った子は、誰だったかな?」間。別の道具「あの子じゃない?」
- **S4 昼間の記憶へ**: サンドバッグ表面に昼の光が重なり、明るい昼のジムへ。子どもが初めてグローブをつけている。手はまだ小さく、グローブが少し大きい。
- **S5 不安**: 子が練習の途中で止まる。少し不安そうに保護者を見る。急かさない・怒鳴らない・比較しない。インストラクターも無理に動かさない。
- **S6 親のうなずき**: 保護者が言葉なく静かにうなずく。「大丈夫」が伝わる。子は親の顔を見て少し安心。「親の背中を見て育つ」を説明でなく視線で表現。
- **S7 もう一度の一歩**: 子が前を向く。大技・強打は不要。小さく一歩、ミットに軽くパンチ。インストラクターが笑顔で受け止める。正確さより「もう一度やってみたこと」。
- **S8 笑顔**: 練習が終わる。少し汗をかいて笑う。親のもとへ戻る。派手な成功ポーズ不要。小さな満足と安心。
- **S9 手**: グローブを外す。小さな手が見える。親の手がタオルを渡す。一瞬、手と手が触れる。必要ならごく短く少し成長した手を重ねてもよいが、説明的な成長モンタージュにしすぎない。
- **S10 帰り道**: 子と親がジムを出る。手をつないでも、子が少し前を歩き親が背中を見る構図でも。日常へ戻る温かな空気。
- **S11 再び夜**: 夜のジムへ戻る。道具たちが昼を思い出す。「何度もお母さんを見ていたね」「でも、最後は笑って帰ったね」小さく「また少し強くなったね」。ここでの「強さ」は勝つ強さでなく、怖さを少し越えたこと。
- **S12 ラスト**: 照明がゆっくり消える。道具が「また明日、みんなに会えるかな」。暗転。FLATUPロゴ。コピーは1〜2個に絞る(例: 人生の最初の一歩に寄り添う場所。/ 世界一初心者に優しい格闘技ジム。)。最後に「体験受付中」または「はじめの一歩を、見に来ませんか。」を静かに表示。

## 18. ナレーションの方向性

説明口調にしない。落ち着き・温かさ・押しつけない声。親を責めない。感動させようと力まない。余白を残す。

## 20. ビジュアルの方向性

実際の雰囲気を大切に: 白基調の清潔な空間 / 明るい白壁 / 白い床 / 自然光 / 木目 / 観葉植物 / 温かい照明 / 威圧感のない配置 / 親しみやすい用品 / 明るく安全 / 実在しそうな空間 / 子ども・大人に安心。蛍光ピンクや蛍光イエローの特徴的サンドバッグはブランドの個性として使えるが、派手で攻撃的な格闘技空間に見せない。

## 21. アニメーションの方向性

## 22. カメラと編集

## 23. 音楽と音響

感動を押しつけない。推奨: 静かなピアノ / 柔らかいストリングス / 木の音 / 夜の空調音 / 時計の針 / グローブが棚に触れる小さな音 / 子どもの呼吸 / ミットに触れる軽い音 / 遠くの笑い声 / 帰り際のドアの音。音楽だけで泣かせない。無音・環境音の時間を作り、視聴者が自分や自分の子を重ねられる余白を残す。

## 24. 媒体と制作方法

Instagramリール / TikTok / YouTubeショート / LINE VOOM 想定。基本は縦型。
推奨仕様: 9:16 / 第一候補30〜45秒 / 短縮15秒 / 必要なら60秒 / 冒頭1〜3秒で静かな違和感 / 字幕なしでも感情が伝わる / 音を出せない視聴者向け字幕版 / ラストにロゴと体験導線。
制作方法(1つの長文から一括生成しない): ①シーンごとの静止画を作る ②キャラ・道具・内装の一貫性を確認 ③各静止画をImage-to-Videoで短く動かす ④編集でつなぐ ⑤音・会話・ナレーション・字幕を追加。**1つの動画生成につき1つの明確な動作**を基本に。
