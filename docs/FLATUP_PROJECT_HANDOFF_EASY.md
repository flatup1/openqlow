## 大事な"方向転換"が3つ

1. **完全オリジナルでいく**
   - 有名アニメのマネはしない。作品名もプロンプトに書かない。
   - 「おもちゃに命が宿ったような、あたたかい3Dアニメ風」で表現する。

2. **ホームページを作り直す前に、まず"広告"を作る**
   - いきなり全部変えると大変だから、先に短い広告を作って反応を見る。
   - **今いちばんやること = 広告 第1話をちゃんと作れる形にすること。**

3. **中身は"格闘技"じゃなくて"親の気持ち"を描く**
   - 見てほしい人は、4〜12歳の子どもがいる親。
   - 「今しかない、子どもとの時間って大事だよね」と、そっと思ってもらう。

## 作りたい動画のキモチの流れ

不安 → きぼう → 安心 → 小さな挑戦 → うれしい → やさしさをつなぐ → 行動(体験予約)

**しかけ(かくし味):**
- 「親の背中を見て、子は育つ」を、**言葉で説明せず映像で見せる**。
  (子が親のマネをする場面を積み上げて、最後に一言だけ言う)
- **夜のジムで、道具たち(ミット・グローブ・サンドバッグ)がしゃべる**。
  その日がんばった子の話を、やさしく語り合う。童話みたいな雰囲気。
